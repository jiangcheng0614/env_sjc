# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1732845020131-071ac58b-d296-4f80-8a02-def629bb1750.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
请选择存放数据文件的文件夹

**成交类型：**  
选择成交类型，例：整体成交

**日期控件类型：**  
下拉选择日期控件类型  
支持：近1天，近7天，近30天，自然日，自然周，自然月，自定义，大促

**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd；若选择自定义、日期范围输入框，请补充一个至多连续31天的日期范围，格式：yyyy-mm-dd_yyyy-mm-dd；若选择大促，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字词

**<font style="color:#DF2A3F;">下载类型：</font>**<font style="color:#DF2A3F;">  
</font><font style="color:#DF2A3F;">请选择下载文件的类型，现有下载当前明细以及直接下载两项。默认为下载当前明细(即二次下载)</font>

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

## 2.输出参数
**文件保存路径：**  
在此处接收文件保存路径
