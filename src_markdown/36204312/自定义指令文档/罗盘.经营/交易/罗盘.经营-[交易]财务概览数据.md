# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1677759751587-d65566bb-abfb-4bea-8a32-38c831a53c34.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
选填项。请选择存放数据文件的文件夹，如不填写表示不输出Excel文件

**日期控件类型：**  
下拉选择日期控件类型  
支持：近7天，近30天，近90天，自然日，自然周，自然月

**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd

**账户类型：**  
填写您需要获取数据的账户类型（如不填写则不做该项选择操作）

全部/聚合账户/微信/支付宝/合众支付...

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

**网页加载重试次数：**

如果单次网页加载卡顿，重试加载网页的次数，默认不重试

## 2.输出参数
**财务概览数据：**  
返回值为字典类型，因为是聚合数据，所以这里只提供相关数据列表，键为{'贷款总览', '贷款账户收支概况'}

```json
{
  '贷款总览': [[表头行], [数据行], [数据行], ... ...],
  '贷款账户收支概况': [[表头行], [数据行], [数据行], ... ...],
}
```

**文件保存路径：**  
如果填写了存储目录，可在此处接收文件保存路径
