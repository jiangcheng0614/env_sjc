# 一.功能描述
用于获取 罗盘·经营-短视频-视频榜单-热门短视频榜单 的数据。

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1677740056506-642e8998-6bab-4c10-8b0c-79f22b3f5794.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
选填项。请选择存放数据文件的文件夹，如不填写表示不输出Excel文件

**日期控件类型：**  
下拉选择日期控件类型  
支持：日榜，周榜，月榜，大促榜

**具体日期：**  
若选择日、周、月榜，请补充一个具体日期，格式：yyyy-mm-dd；若选择大促榜，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字词

**一级标签：**  
填写您需要获取数据的一级标签（如不填写则不做该项选择操作）

输入完整的一级标签

**二级标签：**  
填写您需要获取数据的二级标签（如不填写则不做该项选择操作）

输入完整的二级标签，或目标选项唯一包含的关键字词

**账号类型：**  
填写您需要获取数据的账号类型（如不填写则不做该项选择操作）

自营账号/带货账号...

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

**网页加载重试次数：**

如果单次网页加载卡顿，重试加载网页的次数，默认不重试

## 2.输出参数
**热门短视频榜单数据：**  
返回值为字典类型，内容为{'数据列表': 解析后的数据列表, '数据字典': 解析后的数据字典}

```json
{
  '数据列表': [
    [表头行],
    [数据行],
    [数据行],
    ... ...
  ]
  '数据字典': {
    '主键1': {
      '字段1': 数据1,
      '字段2': 数据2,
      ... ...
    },
    '主键2': {
      '字段1': 数据1,
      '字段2': 数据2,
      ... ...
    },
    ... ...
  }
}
```

**文件保存路径：**  
如果填写了存储目录，可在此处接收文件保存路径
