# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1719194683816-3bb0fae7-e912-4240-b37a-7a4217b35200.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
请选择存放数据文件的文件夹。

**日期控件类型：**  
下拉选择日期控件类型  
支持：近1天，近7天，近30天，自然日，自然周，自然月，大促

**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd；若选择自定义、日期范围输入框，请补充一个至多连续31天的日期范围，格式：yyyy-mm-dd_yyyy-mm-dd；若选择大促，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字词

**合作账号：**  
填写您需要获取数据的合作账号（如不填写则不做该项选择操作）

输入完整的合作账号名称，或目标选项唯一包含的关键字词

**视频范围：**  
填写您需要获取数据的视频范围（如不填写则不做该项选择操作）

全部视频/新发布视频/历史视频...

**短视频名称：**

填写您要查询特定短视频的名称

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

## 2.输出参数
**文件保存路径：**  
在此处接收文件保存路径
