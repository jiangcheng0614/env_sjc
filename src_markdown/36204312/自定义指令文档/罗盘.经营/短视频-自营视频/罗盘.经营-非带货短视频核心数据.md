# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/25990797/1677747201481-9a60fb6e-39c5-4793-b553-cba4f6c98a10.png)

## 1.输入参数
**网页对象：**  
下拉选择当前页面变量，当前页面不一定要在这个界面，但是抖店必须处于登录的状态。



**日期控件类型：**  
下拉选择日期控件类型。  
支持：“近1天”、“近7天”、“近30天”、“自然日”、“自然周”、“自然月”、“大促”。



**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd；若选择自定义、日期范围输入框，请补充一个至多连续31天的日期范围，格式：yyyy-mm-dd_yyyy-mm-dd；若选择大促，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字。



**存储目录：**  
选填项，请选择存放数据文件的文件夹，如不填写表示不输出Excel文件。

## 2.输出参数
**文件保存路径：**  
如果填写了 存储目录，本参数返回输出文件路径。



**核心数据：**  
爬取的数据，格式:

```json
{
	"短视频观看次数": 308434,
	"引流直播间次数": 5242,
	"点击关注次数": 252,
	"点赞次数": 456,
	"评论次数": 12
}
```
