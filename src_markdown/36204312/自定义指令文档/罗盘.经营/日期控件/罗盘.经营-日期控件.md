# 一.功能描述
用于 罗盘·经营平台 的日期选择。

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1678243884248-2414dbba-df55-47e7-ae18-4e69371bb380.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**按钮文本：**  
您选择的要操作的对应按钮上的文本，比如：“近7天” “自然日” “日榜” “大促” “自定义”等；如存在“更多”按钮，请输入“更多”按钮内需要选择的按钮的文本，而请勿填写“更多”；如需操作填写日期范围输入框，请输入“日期范围输入框”。

目前支持：实时、近n天、分钟榜、小时榜、今日榜、昨日、日榜、自然日、周榜、自然周、月榜、自然月、大促榜、大促、自定义、日期范围输入框。

**开始日期：**  
选填项，若需提供具体日期（例：日榜、自然日、自然周、自然月 ...），请填写此项。日期格式：yyyy-mm-dd

**结束日期：**  
选填项，若需提供日期范围（例：自定义、日期范围输入框 ...），请填写此项。日期格式：yyyy-mm-dd（自定义支持至多连续31天的日期选择）

**大促名称：**  
选填项，请填写大促选项名称（大促名称需与平台对应选项内文字保持一致，或您的目标选项唯一包含您填写的关键字词）

## 2.输出参数
**控件类型：**  
本次判断出的控件类型（操作模式）
