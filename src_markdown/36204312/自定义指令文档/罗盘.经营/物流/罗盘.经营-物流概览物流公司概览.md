# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/25990797/1677747782015-5309f2d9-d612-4a5d-b246-a8f0615ef3b6.png)

## 1.输入参数
**网页对象：**  
下拉选择当前页面变量，当前页面不一定要在这个界面，但是抖店必须处于登录的状态。



**存储目录：**  
选填项，请选择存放数据文件的文件夹，如不填写表示不输出Excel文件。

## 2.输出参数
**文件保存路径：**  
如果填写了 存储目录，本参数返回输出文件路径。



**物流公司概览数据：**
