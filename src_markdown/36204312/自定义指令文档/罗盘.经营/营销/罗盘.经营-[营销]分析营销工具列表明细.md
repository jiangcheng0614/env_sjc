# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1677761186795-ef8d256e-eab9-4064-9620-131618cd2e97.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
请选择存放数据文件的文件夹

**日期控件元素：**  
传入您需要点击的单个日期控件按钮

因为该页面存在多组日期控件，所以需要指定目标日期控件元素操作，比如您需要进行自然日操作，就捕获并传入目标自然日按钮元素

**日期控件类型：**  
下拉选择日期控件类型  
支持：近7天，近30天，近90天，自然日，自然周，自然月

**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd

**营销工具类别：**  
填写您需要获取数据的营销工具类别（如不填写则不做该项选择操作）

优惠券/限时限量购/满减...

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

## 2.输出参数
**文件保存路径：**  
在此处接收文件保存路径
