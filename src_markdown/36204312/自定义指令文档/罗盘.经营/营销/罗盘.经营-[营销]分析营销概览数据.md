# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1677760988550-0a17187f-ec3e-41d5-8f98-ccfcd921148a.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
选填项。请选择存放数据文件的文件夹，如不填写表示不输出Excel文件

**日期控件类型：**  
下拉选择日期控件类型  
支持：昨日，近7天，近30天

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

**网页加载重试次数：**

如果单次网页加载卡顿，重试加载网页的次数，默认不重试

## 2.输出参数
**营销分析概览数据：**  
返回值为字典类型，内容为{'数据列表': 解析后的数据列表, '数据字典': 解析后的数据字典}

```json
{
  	'数据列表': [
        [表头行],
        [数据行],
        [数据行],
        ... ...
    ]
    '数据字典': {
        '主键1': {
            '字段1': 数据1,
        		'字段2': 数据2,
            ... ...
        },
        '主键2': {
            '字段1': 数据1,
        		'字段2': 数据2,
            ... ...
        },
        ... ...
    }
}
```

**文件保存路径：**  
如果填写了存储目录，可在此处接收文件保存路径
