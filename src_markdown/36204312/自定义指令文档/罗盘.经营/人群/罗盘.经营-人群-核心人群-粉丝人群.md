## 1.输入参数
**网页对象：**  
下拉选择当前页面变量，当前页面不一定要在这个界面，但是抖店必须处于登录的状态。



**存储目录：**  
选填，填写下载明细文件存储的文件夹路径，不填写，只返回字典。  
格式：文件夹路径。  
输入样式："C:\Users\SHADOWBOT\Desktop\新建文件夹"



**日期控件类型：**  
下拉选择日期控件类型，选择对应即可  
支持：“昨日”、“近7天”、“近30天”。



**账号选择：**  
填写，粉丝人群下方的账号下拉框选项。  
例如：全部账号  


### **高级**
**全局超时时间：**  
输入一个**整数**，修改指令里面等待元素超时时间等  
例如：如下等待对话框出现超时时间

## 2.输出参数
**结果字典：**

```json
{
	'内容触达用户': ['0', '较上周期下降0'],
	'内容兴趣用户': ['0', '较上周期下降0'],
	'新人粉丝': ['0', '较上周期下降0'],
	'忠诚粉丝': ['0', '较上周期下降0']
}
```



**文件保存路径：**  
输出文件下载后的路径。  
例如:C:\Users\SHADOWBOT\Desktop\新建文件夹\罗盘.经营-人群-核心人群-粉丝人群_昨日_20230224165432.xlsx
