## 1.输入参数
**网页对象：**  
下拉选择当前页面变量，当前页面不一定要在这个界面，但是抖店必须处于登录的状态。



**存储目录：**  
选填，填写下载明细文件存储的文件夹路径，不填写，只返回字典。  
格式：文件夹路径。  
输入样式："C:\Users\SHADOWBOT\Desktop\新建文件夹"

### **高级**
**全局超时时间：**  
输入一个**整数**，修改指令里面等待元素超时时间等  
例如：如下等待对话框出现超时时间

## 2.输出参数
**结果字典：**

```json
{
	'物流负向反馈率': ['0%较前一日上升0%', '超过同行同规模0%商家'],
	'揽收及时率': ['0%较前一日下降0%', '超过同行同规模0%商家'],
	...
}
```



**文件保存路径：**  
输出文件下载后的路径。  
例如:C:\Users\SHADOWBOT\Desktop\新建文件夹\罗盘经营_服务_商家体验分_分析诊断__20230224171309.xlsx
