# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1677749911022-aa7bdf5e-6c27-48ae-a379-524b485ad467.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保罗盘·经营or抖店处于登录状态）

**存储目录：**  
选填项。请选择存放数据文件的文件夹，如不填写表示不输出Excel文件

**日期控件类型：**  
下拉选择日期控件类型  
支持：近1天，近7天，近30天，自然日，自然周，自然月，大促

**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd；若选择自定义、日期范围输入框，请补充一个至多连续31天的日期范围，格式：yyyy-mm-dd_yyyy-mm-dd；若选择大促，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字词

**内容：**  
填写您需要获取数据的品牌商家（如不填写则不做该项选择操作）

全部/带货/非带货...

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

**网页加载重试次数：**

如果单次网页加载卡顿，重试加载网页的次数，默认不重试

## 2.输出参数
**行业看后搜视频榜数据：**  
返回值为字典类型，内容为{'数据列表': 解析后的数据列表, '数据字典': 解析后的数据字典}

```json
{
  '数据列表': [
    [表头行],
    [数据行],
    [数据行],
    ... ...
  ]
  '数据字典': {
    '主键1': {
      '字段1': 数据1,
      '字段2': 数据2,
      ... ...
    },
    '主键2': {
      '字段1': 数据1,
      '字段2': 数据2,
      ... ...
    },
    ... ...
  }
}
```

**文件保存路径：**  
如果填写了存储目录，可在此处接收文件保存路径
