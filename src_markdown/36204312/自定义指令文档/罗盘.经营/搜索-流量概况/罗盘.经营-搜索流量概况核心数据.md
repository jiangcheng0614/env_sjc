# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/25990797/1677747695738-a9cc6105-b640-488a-9336-03621572c846.png)

## 1.输入参数
**网页对象：**  
下拉选择当前页面变量，当前页面不一定要在这个界面，但是抖店必须处于登录的状态。



**日期控件类型：**  
下拉选择日期控件类型。  
支持：“近1天”、“近7天”、“近30天”、“自然月”、“大促”。



**具体日期：**  
若选择自然日、周、月，请补充一个具体日期，格式：yyyy-mm-dd；若选择自定义、日期范围输入框，请补充一个至多连续31天的日期范围，格式：yyyy-mm-dd_yyyy-mm-dd；若选择大促，请补充一个完整的大促选项名称，或目标大促选项唯一包含的关键字。



**成交类型：**  
对应页面上的成交类型下拉控件。



**存储目录：**  
请选择存放数据文件的文件夹。

## 2.输出参数
**文件保存路径：**  
输出文件路径。
