# 描述
发送本地文件至群会话，这个指令相当于【上传文件】和【发送在线文件至群会话】两个指令的结合

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1684808255847-eb8ff090-043a-410b-9688-92f885c167f0.png)



# 参数配置
## 指令输入
### 常规
**钉盘连接对象：**建立钉钉云盘连接指令获取的对象

**空间ID：**文件所在钉盘空间 ID

**群会话ID：**钉钉群会话 ID

**本地文件路径：**选择发送的文件路径
