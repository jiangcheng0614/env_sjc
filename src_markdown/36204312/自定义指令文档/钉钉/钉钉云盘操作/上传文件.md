# 参数配置
## 指令输入
### 常规
**钉盘连接对象：**建立钉钉云盘连接指令获取的对象

**空间ID：**文件所在钉盘空间 ID

**父目录ID：**文件所在钉盘中的文件夹 ID

**本地文件路径：**待上传的本地文件路径

### 高级




## 指令输出
**返回文件ID至：**钉盘中文件的唯一标识，可用于后续的发送文件至群会话
