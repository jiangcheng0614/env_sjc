描述

通过应用身份发送本地文件至指定用户，这个指令相当于【上传文件】和【发送在线文件至用户】两个指令的结合

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1698810078369-29dcf62f-33d1-4bbe-b603-38e556955e98.png)

# 参数配置
## 指令输入
### 常规
**钉盘连接对象：**建立钉钉云盘连接指令获取的对象

**空间ID：**文件所在钉盘空间 ID

**父目录ID：**用于上传文件存储的文件夹 ID

**用户ID：**指定用户的 ID

**本地文件路径：**待发送的文件路径

### 高级
无
