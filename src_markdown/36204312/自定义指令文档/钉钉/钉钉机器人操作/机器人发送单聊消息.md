# 描述
通过钉钉机器人发送单聊消息

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1695008765480-56a13992-0728-407a-a8d9-d7c16151cadd.png)



# 参数配置
## 指令输入
### 常规
**钉钉机器人对象：**通过【建立钉钉机器人连接】指令获取

**用户ID：**接受消息的用户ID，多个用户用列表形式:["id1", "id2"]。最多20个用户

**消息模板**：发送的消息模板

+ 文本类型：发送文本类型消息
+ Markdown类型：发送Markdown类型消息，可以在高级页设置title
+ 图片类型：在线图片地址
+ 文件类型：本地文件地址

**消息内容**：发送的消息内容

### 高级
**消息标题：**选择消息模板为Markdown类型，此参数才会生效。会在钉钉消息列表显示title

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1695009144089-cc460997-7003-4754-af7d-f17fd73f3f66.png)

## 指令输出
**无效的用户ID列表**：无效的用户ID列表，可以检查一下ID是否输入正确，不要有多余空格

**被限流的用户ID列表**：被限流的用户ID列表
