# 一.功能描述
根据指令“获取转换字典”得到的字典，进行原始数据转换。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1682564817679-a7b9836b-6e09-4ade-805c-153a7f396227.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1682565457508-aea93eca-08c3-4bc8-91db-dc3fc7c2ccc0.png)

## 1.输入参数
**原始数据：**

输入从页面直接抓取的元素文本原始数据，支持字符串、单字符。

****

**转换字典：**

指令“获取转换字典”得到的字典。

## 2.输出参数
**转后数据：**  
以字典形式输出加密文字和正常文字之间的对应关系。
