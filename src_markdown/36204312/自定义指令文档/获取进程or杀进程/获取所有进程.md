# 一.功能描述
本指令可获取本机所有正在运行的<font style="background-color:#FBDE28;">进程名称</font>以及相应的<font style="background-color:#FBDE28;">进程号</font>

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/561060/1675922084036-087a031d-5eee-4910-90bb-6b263f98d0e6.png)

## 1.输入参数
无

## 2.输出参数
输出参数为：<font style="background-color:#FBDE28;">所有进程</font>

格式：<font style="background-color:#FBDE28;">字典格式，key为进程号，value为进程名称</font>  
输出样式：<font style="background-color:#FBDE28;">{0: 'System Idle Process', 4: 'System'}</font>

点击使用即可，进程名称及进程号会展示在日志中，如有需要后续可自定义添加后续操作，储存在文件中。
