# 描述
在旺店通启动后，该指令完成旺店通的登录动作

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1682595477491-d0d78d1b-4a3b-4708-ab63-25b4dcba8c69.png)



# **输入**
### **用户名：**
旺店通账号

### 密码：
旺店通密码



# 输出
### proce_resul:
登录成功返回 True，登录失败返回 False
