##### 概述
当前版本指令支持淘宝联盟商家中心-数据分析下推广概览和商品分析的日期自定义选择及快捷日期选择

##### 参数定义
+ **web_page**
    - 类型：网页对象
    - 下拉选择淘宝联盟商家中心-数据分析推广概览或商品分析所在页面网页对象
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择

```python
  快捷日期列表 = ['今日实时','昨日','7天','近15天','近30天']
```
