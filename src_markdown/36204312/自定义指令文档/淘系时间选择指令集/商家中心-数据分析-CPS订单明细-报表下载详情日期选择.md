##### 概述
本指令实现了淘宝联盟商家中心_数据分析_CPS订单明细_报表下载详情页面日期的自定义选择，支持自定义输入及快捷日期输入。

##### 指令配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208055996-c65778b1-3a92-43bf-8bf7-8544d0b64c8d.png)

##### 操作页面
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208056700-d28fa0d9-a72c-4743-b97e-2313b8f3ce72.png)



+ **web_page**
    - 类型：网页对象
    - 下拉选择商家中心_数据分析_CPS订单明细_报表下载详情所在页面网页对象（指令会自动点击时间选择控件，到达下载报表详情页面即可调用本指令进行日期的自定义选择）
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd hh:mm:ss~yyyy-mm-dd hh:mm:ss，例：2022-11-12 00:00:00~2022-12-12 00:00:00
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择.

```python
  快捷日期列表 = ['今天','昨天','过去7天','过去15天','过去30天]
```
