##### 概述
本指令实现了淘宝联盟商家中心_数据分析_类目分析页面日期的自定义选择，支持自定义输入及快捷日期输入。

##### 指令配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208077277-487e59ec-0034-4bef-84d3-826a887ad009.png)

##### 操作页面
##### ![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208077706-f50776aa-b28d-4c73-a15f-b9759aeb9c9c.png)
+ **web_page**
    - 类型：网页对象
    - 下拉选择商家中心_数据分析_定向计划报表所在页面网页对象
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择

```python
  快捷日期列表 = [近7天','近15天','近30天']
```
