##### 概述
本指令实现了淘宝联盟商家中心_数据分析_服务商合作报表下的**普通招商报表**，**私域招商报表**，**内容招商报表**，以及自主推广报表下**营销计划日常报表**和**营销计划整体报表**页面日期的自定义选择，支持自定义输入及快捷日期输入。

##### 操作页面
##### ![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208071449-7d56c160-7618-437f-b6c9-8757cc8ec91e.png)
+ **web_page**
    - 类型：网页对象
    - 下拉选择商家中心_数据分析_服务商合作报表下的**普通招商报表**，**私域招商报表**，**内容招商报表**，以及自主推广报表下**营销计划日常报表**和**营销计划整体报表**等页面的任一网页对象
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择

```python
  快捷日期列表 = ['今日实时','昨天','上月','最近7天','最近15天','最近30天']
```
