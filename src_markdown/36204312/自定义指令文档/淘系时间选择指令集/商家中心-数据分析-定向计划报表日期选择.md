##### 概述
本指令实现了淘宝联盟商家中心_数据分析_定向计划报表_日期的自定义选择，支持自定义输入及快捷日期输入。

##### 指令配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208065737-d141207a-b759-400d-b36c-b616f0054727.png)

##### 操作页面
##### ![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208066179-e107ae46-c4be-4630-b7d2-062349927621.png)
+ **web_page**
    - 类型：网页对象
    - 下拉选择商家中心_数据分析_定向计划报表所在页面网页对象
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择

```python
  快捷日期列表 = ['今日','昨日','过去7天','过去15天','过去30天','过去60天','过去90天']
```
