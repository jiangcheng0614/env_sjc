##### 概述
本指令实现了淘宝联盟家中心_数据分析_自选计划报表及通用计划报表页面日期的自定义选择，支持自定义输入及快捷日期输入。

##### 指令配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208085820-599bfbbe-8d66-4bc6-bfe1-bef2afe0cc04.png)

##### 操作页面
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677208086265-4907f9c2-a421-4de4-a0ff-1516a5dfb34b.png)



+ **web_page**
    - 类型：网页对象
    - 下拉选择商家中心_数据分析_定向计划报表所在页面网页对象
+ **日期**：
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择，**注意：中文与空格之间的空格不可省略，否则会无法完成快捷日期的选择**

```python
  快捷日期列表 = ['昨日','过去 7 天','过去 15 天','过去30天','过去 60 天','过去 90 天']
```
