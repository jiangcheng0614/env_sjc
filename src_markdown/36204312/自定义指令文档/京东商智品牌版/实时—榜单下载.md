# **输入**
### **web_page：**
任意一个已登录京东商智品牌版后台的网页对象

### 当前日期范围：
下拉选择：

今日累计、昨日晚8累计、昨日累计、28h回放、昨日晚8累计（农历同比）

### 对比时间：
填 YYYY-MM-DD（最早一年前，但选不到同一天）， 或快捷日期【环比昨日】、【同比】 、【农历同比】

### 保存文件夹：
下载文件保存的文件夹路径

### 文件名：
下载后的文件名，要带拓展名

# 输出
### 无
