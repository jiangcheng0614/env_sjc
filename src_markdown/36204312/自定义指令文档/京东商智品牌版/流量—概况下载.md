# 描述	
点击【自定义】选择时间，点击【对比时间】选择时间，点击【下载数据】

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1698740563413-908a1661-a843-4e62-9064-fc1754f33983.png)



# **输入**
### **web_page：**
任意一个已登录京东商智品牌版后台的网页对象

### 开始时间：
YYYY-MM-DD

### 结束时间：
填 YYYY-MM-DD（最早一年前，但选不到同一天）， 或快捷日期【环比昨日】、【同比】 、【农历同比】

### 开始对比时间：
YYYY-MM-DD，或快捷对比【环比】、【同比】

### 结束对比时间：
YYYY-MM-DD

### 保存文件夹：
下载文件保存的文件夹路径

### 文件名：
下载后的文件名，要带拓展名

# 输出
### 无
