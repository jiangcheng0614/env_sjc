# 描述

# **输入**
### **web_page：**
任意一个已登录京东商智品牌版后台的网页对象

### 开始时间：
YYYY-MM-DD

### 结束时间：
YYYY-MM-DD

### 下载维度：
SKU、SPU

### 展现方式：
汇总展示

按天展示

地域销售-汇总展示

地域销售-按天展示

### 保存文件夹：
下载文件保存的文件夹路径

### 文件名：
下载的文件名，不要带拓展名

### 品牌选择：
默认全部

选择品牌则切换python输入方式，例如['苏','飞']

# 输出
### 无
