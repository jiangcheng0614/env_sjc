# **输入**
### **web_page：**
任意一个已登录京东商智品牌版后台的网页对象

### 数据类型：
实时或离线

### 开始时间：
数据类型是【实时】时，填【今日累计】或【昨日累计】

数据类型是【离线】时，填 YYYY-MM-DD

### 结束时间：
YYYY-MM-DD，数据类型是【实时】时不用填

### 对比时间：
【环比】、【同比】，数据类型是【实时】时，额外可填 YYYY-MM-DD或【农历同比】

### 指标：
<font style="background-color:rgb(242, 248, 254);">浏览量(PV)</font>

访客数(UV)

人均浏览量(人均PV)

平均停留时长(秒)

UV价值

客单价

加购客户数

加购转化率

成交客户数

成交转化率

成交金额

<font style="background-color:rgb(246, 247, 251);">成交子单量</font>

### 保存文件夹：
下载文件保存的文件夹路径

### 文件名1：
下载后的流量去向文件名，要带拓展名

### 文件名2：
下载后的退商详页文件名，要带拓展名

# 输出
### 无
