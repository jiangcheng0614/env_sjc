# 描述	
点击【自定义】选择时间，点击【下载数据】

# **输入**
### **web_page：**
任意一个已登录京东商智品牌版后台的网页对象

### 当前日期开始、当前日期结束：
YYYY-MM-DD，（时间跨度为2天，前天~昨天、或昨天~今天）

### 当前时间开始、当前时间结束：
小时 00~23，

### 对比时间：
填 YYYY-MM-DD（最早一年前，但选不到同一天）， 或快捷日期【同比】 、【农历同比】

### 保存文件夹：
下载文件保存的文件夹路径

### 文件名：
下载后的文件名，要带拓展名

### 下载类型：
下拉选择：

汇总下载

分小时下载

累计下载

# 输出
### 无
