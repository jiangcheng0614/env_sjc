# 指令描述
该指令实现jde系统的第三方查询数据下载

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707271543875-b31131ad-079c-4ca0-9193-4935df363602.png)

**输入参数 : **

**保存文件夹路径**: 非必输；默认保存到c盘

**保存文件名称**：非必输；默认按"第三方查询_查询开始日期_结束日期.xls"保存

**查询开始日期**：非必输；数据查询开始日期，如不传默认按系统当前日期查询

**查询开始日期**：非必输；数据查询结束日期，如不传默认按系统当月底日期查询

**输出参数 : **

**第三方查询返回布尔值**：返回下载结果

**文件路径：**返回结果文件本地路径
