# 指令描述
该指令实现jde系统的卡手续费数据下载

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707274102804-6bddd5ee-6b94-45f6-ac85-9ff5dfaacec8.png)

**输入参数 : **

**保存文件夹路径**: 非必输；默认保存到c盘

**保存文件名称**：非必输；默认按"第三方查询_查询开始日期_结束日期.xls"保存

**查询开始日期**：非必输；数据查询开始日期，如不传默认按当前日期查询

**查询开始日期**：非必输；数据查询结束日期，如不传默认按当月底日期查询

**输出参数 : **

**卡手续费下载返回布尔值**：返回下载结果

**文件路径：**返回结果文件本地路径
