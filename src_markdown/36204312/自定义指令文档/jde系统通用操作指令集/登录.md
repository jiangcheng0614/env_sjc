# 指令描述
该指令实现jde系统的登录

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707271194157-cd124e0c-af33-4fe3-9a61-edcaadb2db7a.png)

**输入参数 : **

**网址 **: jde系统登录网址

**账号**：jde系统登录账号

**密码**：jde系统登录密码

**输出参数 : **

**登录返回布尔值**：返回登录结果
