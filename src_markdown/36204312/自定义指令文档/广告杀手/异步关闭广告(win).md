# 描述
后台监测广告弹窗，全流程调用一次即可，随着主流程结束而结束。会根据传入的元素选择器，关闭元素所在的窗口对象

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1690730131522-20374bbf-83cd-4f0c-8cbb-d91b995d6893.png)



# 参数配置
## 指令输入
### 常规
**元素选择器列表：**广告元素列表，可以通过【[组合多元素(win)](https://ydrpa.yuque.com/org-wiki-ydrpa-xtutvv/ga4dm6/list_ele_win)】指令组合元素，再传入到这里





## 指令输出
无
