# 描述
给列表或字典添加进度条功能，返回一个进度条对象

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1693374976131-2f0eb825-a8bf-4d9a-afe7-0f3a3d12895e.png)



# 参数配置
## 指令输入
### 常规
**可迭代对象：**可传入任务列表、字典，也可以传入任务总数

**任务名称**：用来显示在进度条上的任务名称，可任意填写

### 高级
**配置**：用于修改进度条样式，在下方查看如何修改



## 指令输出
**无**

# 修改进度条样式
1.搜索【唤起屏幕保护】指令，点击编辑屏保样式

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1693374850822-c4246519-f659-4c62-a7c5-fdcfa8079ea9.png)



2.编辑好后点击复制

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1693374912044-cb4fdd07-c72a-420c-81e9-42ce2fd12443.png)



3.然后在【初始化进度条】指令的高级页，传给参数：配置。直接粘贴进去即可
