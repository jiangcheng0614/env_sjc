# 一.功能描述
该指令在巨量百应的联盟订单明细页面按时间区间导出数据

![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1714996400450-bc0dc222-80f7-4a70-b5e8-1e4d5a79b32a.png)

## 参数配置
### 指令输入
#### 常规
**网页对象: 当前进行时间选择操作的网页对象**

**时间筛选：订单筛选参数**

**订单状态：订单筛选参数**

**流量渠道：订单筛选参数**

**下单来源：订单筛选参数**

**流量细分渠道：订单筛选参数**

**开始时间：选择订单开始的时间****<font style="color:#DF2A3F;">格式为HH-MM-SS，例(2024-10-10)</font>**

**结束时间：选择订单结束的时间****<font style="color:#DF2A3F;">格式为HH-MM-SS，例(2024-10-10)</font>**

**保存文件夹：保存地址的文件夹名称**

**保存文件名：保存下来的文件夹名**

### **指令输出**
**保存文件名：保存文件夹所在路径**
