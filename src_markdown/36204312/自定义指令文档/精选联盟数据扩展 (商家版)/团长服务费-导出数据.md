# 一.功能描述
该指令在巨量百应的团长服务费明细页面按时间区间导出数据

## ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1715173047805-4898b962-bb21-4b84-b537-0789f449e912.png)参数配置
### 指令输入
#### 常规
**网页对象: 当前进行时间选择操作的网页对象**

**时间筛选：订单筛选参数**

**订单状态：订单筛选参数**

**订单ID：填写订单ID（可为空）**

**下单来源：选择下单来源筛选参数**

**商品ID：填写商品ID(可为空)**

**推广者类型：填写推广者类型来筛选**

**开始时间：选择订单开始的时间****<font style="color:#DF2A3F;">格式为YYYY-MM-DD HH:MM:SS，例(2024-10-10 12:23:45)</font>**

**结束时间：选择订单结束的时间****<font style="color:#DF2A3F;">格式为YYYY-MM-DD HH:MM:SS，例(2024-10-10 12:23:45)</font>**

**保存文件夹：保存地址的文件夹名称**

**保存文件名：保存下来的文件夹名**

### **指令输出**
**保存文件名：保存文件夹所在路径**
