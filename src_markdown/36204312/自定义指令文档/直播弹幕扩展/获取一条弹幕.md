# 描述
该指令实现了获取一条新的弹幕操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1715689348265-5d272860-4167-41d8-8cc8-15cbba598630.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，需要抓取弹幕的网页

**没有弹幕时**：下拉框

+ 一直等待：没有新的弹幕则一直等待，直到有新的弹幕再返回
+ 返回空值：没有新的弹幕直接返回空值







### 高级
****



## 指令输出
**保存弹幕至**：字符串，新的弹幕内容
