# 描述
该指令实现了向网页中添加一个弹幕监听器的操作，帮助后续获取弹幕

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1715688983043-baaff2d8-0a42-454f-9f46-e98a0df9a057.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，需要抓取弹幕的网页

**弹幕块元素**：网页对象，展示弹幕的元素
