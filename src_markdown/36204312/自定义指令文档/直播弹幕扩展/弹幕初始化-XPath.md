# 描述
该指令实现了向网页中添加一个弹幕监听器的操作，帮助后续获取弹幕

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1715689169381-8a8d6cff-d9c1-4a4f-aea9-3c806037b54c.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，需要抓取弹幕的网页

**弹幕相似元素XPath**：字符串，弹幕相似元素，和【弹幕初始化】指令有所不同，这个是捕获弹幕相似元素组，弹幕初始化是捕获一个大的元素块。从而实现更精确的弹幕获取
