# 描述
识别并解析增值税电子发票PDF文件，将结果保存在EXCEL文件中。

![](https://cdn.nlark.com/yuque/0/2023/png/36165419/1690372969774-304e6178-233e-47c4-95d3-60948ccada6b.png)



# 输入
## 电子发票PDF文件:
电子发票PDF文件的路径。

## 结果保存路径:
结果EXCEL保存的路径。

# 输出
当前版本指令暂无输出项

# 流程搭建
直接拖动指令至流程即可

# 运行逻辑
对电子发票的PDF内容进行解析，通过对解析后的数据进行进一步处理，按照电子发票的格式在EXCEL中进行还原。



# Tips：
1、目前只支持此格式的电子发票识别
