# 使用前联系紫鸟开通权限
[紫鸟浏览器API使用前置环境配置](https://ydrpa.yuque.com/org-wiki-ydrpa-xtutvv/ga4dm6/fgell58iew2lkfsr)



# 描述
通过API打开紫鸟上指定的店铺，返回该店铺的网页对象。（打开店铺会自动关闭正在打开的其他店铺）

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1699434875011-c4246f78-2795-42c4-9658-7565ce453c74.png)



# 参数配置
## 输入
**店铺名称：**需要打开的店铺名称





## 输出
**web_page: **打开后的店铺的网页对象
