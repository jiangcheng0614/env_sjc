# 描述
该指令实现了识别轨迹验证码，并进行滑动的操作 **<font style="color:#DF2A3F;">(注:验证码产生费用)</font>**

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1716343009694-86bf5765-0412-4b0e-ab75-ab045ceaa8e6.png)

# 参数配置
## 指令输入
### 常规
**手机连接对象**：手机连接对象，需要进行验证码操作的手机对象

**引擎**：下拉框

+ 影刀引擎 (付费增值服务) ： 走影刀增值扣费
+ 云码引擎 (高级自定义Token)：使用自己的Token

**背景图**：手机元素对象，验证码背景图







### 高级
**云码Token**：字符串，自己的云码Token

**重试次数：**整数，失败重试次数

**偏移距离：**整数，偏移距离



## 指令输出
无
