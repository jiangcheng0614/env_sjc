## 描述
该指令实现京东金融-资金账户部分页面自定义时间的日期选择

![](https://cdn.nlark.com/yuque/0/2023/png/22078351/1677241555869-f2bb1ccb-f065-47b2-bf86-646511663f72.png)

## 配置项说明
**网页对象：**待操作页面的网页对象

**开始日期：**指定的开始日期  例：2020-12-31 23:59:59

**结束日期：**指定的结束日期  例：2023-01-10 23:59:59

**开始日期控件：**待操作的开始日期选择框网页元素

**结束日期控件：**待操作的结束日期选择框网页元素

## 适用页面
+ **交易查询：**全部-自定义时间、收单-自定义时间、付款-自定义时间、退款-自定义时间、分账-自定义时间、POP-自定义时间、融资-自定义时间、缴费单-自定义时间
+ **极速绑卡查询：**自定义时间
+ **商户结算：**收款管理-已收款
+ **账户流水：**实时查询-自定义时间
+ **待缴费单：**自定义时间
+ **审批管理：**代收付审批-自定义时间
+ **调单管理：**自定义时间
+ **证书管理：**自定义时间
+ **下载查询：**预约时间
