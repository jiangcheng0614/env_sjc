# 参数说明
###### 连接对象:必填项，数据库的连接对象
###### 数据库名称:必填项，新建集合的数据库名称
###### 集合名称:必填项，新建集合的名称
###### 筛选语句:必填项，参数为字典，<font style="color:rgb(33, 44, 63);">MongoDB 查询语句通常以字典形式表示。</font>
###### 更新语句:必填项，参数为字典，<font style="color:rgb(33, 44, 63);">MongoDB 查询语句通常以字典形式表示。</font>
