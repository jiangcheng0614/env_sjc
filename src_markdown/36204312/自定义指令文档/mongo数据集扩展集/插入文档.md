# 参数说明
###### 连接对象:必填项，数据库的连接对象
###### 数据库名称:必填项，新建集合的数据库名称
###### 集合名称:必填项，新建集合的名称
###### 文档列表:必填项，需要插入的文档，参数格式是一个列表，需要插入的文档以字典的形式存放在列表中
