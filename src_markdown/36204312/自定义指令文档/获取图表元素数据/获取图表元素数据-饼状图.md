# 描述
通过模拟滑动，获取网页图表元素数据(饼状图)

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1685365426747-b119d80a-d39a-44e3-8975-c9ed8cd3d3d9.png)



# 参数配置
## 指令输入
### 常规
**网页对象：**网页对象

**目标图表元素：**网页中的画布，可视化展示数据的元素

**半径比：**鼠标在画布上画圈圈的半径比,范围: [0~0.99]。需要根据实际情况调整半径长度

**去除空白符**：勾选则会删除所有空白符（包括换行符）

**图表元素置顶：**如果数据元素挡住了图表元素，则需要勾选此项



### 高级
**数据元素：**虽然指令有自动定位数据元素(悬浮移动展示数据的元素)的功能，但并不能涵盖所有情况。这时候需要手动到网页中找到该元素，并复制它的xpath路径和css选择器传递到这里

**移动速度：**调整移动的速度

**偏移：**有些情况鼠标必须低于该值所展示的区域才能显示数据，这个时候就需要修改一下鼠标的位置，范围[-1,1]





## 指令输出
**保存数据列表至：**监听到的数据
