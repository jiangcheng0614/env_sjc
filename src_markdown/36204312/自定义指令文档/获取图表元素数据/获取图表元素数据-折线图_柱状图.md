# 描述
通过模拟滑动，获取网页图表元素数据

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1685364935395-ff860238-ead6-4e0c-8dde-35e6fd79bc06.png)



# 参数配置
## 指令输入
### 常规
**网页对象：**网页对象

**目标图表元素：**网页中的画布，可视化展示数据的元素

**去除空白符**：勾选则会删除所有空白符（包括换行符）

**图表元素置顶：**如果数据元素挡住了图表元素，则需要勾选此项



### 高级
**数据元素：**虽然指令有自动定位数据元素(悬浮移动展示数据的元素)的功能，但并不能涵盖所有情况。这时候需要手动到网页中找到该元素，并复制它的xpath路径和css选择器传递到这里

**移动方向：**根据实际情况选择，从左到右、从上到小

**移动速度：**调整移动的速度

**移动模式：**调整移动的模式

**移动步长：**改变每次移动的步长

**偏移：**有些情况鼠标必须低于该值所展示的区域才能显示数据，这个时候就需要修改一下鼠标的位置，范围[-1,1]



当X轴数据比较密集的时候，如果移动太快会导致数据抓取不全，建议用以下方案：

针对X轴数据比较密集的选择：慢、抖动、短

一般情况的选择：快、平滑、长





## 指令输出
**数据元素：**如果自动定位元素失败，则需要手动传入数据元素（承载数据的元素），仅支持xpath和css

**保存数据列表至：**监听到的数据
