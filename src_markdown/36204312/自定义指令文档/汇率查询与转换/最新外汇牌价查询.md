# 一.功能描述
查询当前最新的指定币种的外汇牌价（即100元的指定币种的指定价格类型所对应的人民币价格）。

![](https://cdn.nlark.com/yuque/0/2023/png/27512628/1678191574037-806ac07e-e10a-4031-85df-764b7f200381.png)

# 二.使用参数说明
### 输入：
币种：（下拉选择列表）选择想要查询的币种  

_<font style="color:#8A8F8D;">默认：美元</font>_

价格类型：（下拉选择列表）选择想要查询的货币价格类型

_<font style="color:#8A8F8D;">可选项有：中行折算价、现汇买入价、现钞买入价、现汇卖出价、现钞卖出价。默认：中行折算价</font>_

### 输出：
查询结果：类型为字符串，输出查询到的结果

_<font style="color:#8A8F8D;">注：如果想用查询到的结果进行计算，需要将其转化为浮点数（如：float(currency_rate)）</font>_

# 四.常见问题
Q1：如果查询到的结果为 - 是什么原因？（如下图所示）

![](https://cdn.nlark.com/yuque/0/2023/png/27512628/1678193822683-1ca14033-de4b-4f3c-a8c3-e2f2d717dba2.png)

A1：这是由于该币种的该价格在中国银行网站上确实查询不到对应价格数值，所以返回值为 - ，这个不是因为指令本身的问题。
