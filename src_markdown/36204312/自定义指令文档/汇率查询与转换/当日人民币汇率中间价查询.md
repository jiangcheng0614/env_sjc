# 一.功能描述
查询当日银行间外汇市场人民币汇率中间价信息（数据来源为当日中国人民银行早9点公布的数据）。

![](https://cdn.nlark.com/yuque/0/2023/png/27512628/1678241075485-98f0a5e8-09c7-41e2-909d-b88e311dc45c.png)

# 二.使用参数说明
### 输入：
币种：（下拉选择列表）选择想要查询的币种

_<font style="color:#8A8F8D;">默认：美元</font>_

兑换方式：（下拉选择列表）选择想要查询的兑换方式

_<font style="color:#8A8F8D;">可选项有：所选币种对人民币、人民币对所选币种。默认：所选币种对人民币</font>_

### 输出：
查询结果：类型为字符串，输出查询到的结果

_<font style="color:#8A8F8D;">注：查询结果为一个等式，例：1美元=6.9525元人民币、1元人民币=0.14383美元</font>_
