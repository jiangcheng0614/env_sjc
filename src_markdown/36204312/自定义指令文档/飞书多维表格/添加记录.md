## <font style="color:rgb(44, 62, 80);">参数配置</font>
### <font style="color:rgb(44, 62, 80);">指令输入</font>
**<font style="color:rgb(44, 62, 80);">多维表格对象:</font>**_<font style="color:rgb(44, 62, 80);">多维表格对象</font>_<font style="color:rgb(44, 62, 80);">; 由建立多维表格连接指令返回的多维表格对象</font>

**<font style="color:rgb(44, 62, 80);">指定数据表方式:</font>**_<font style="color:rgb(44, 62, 80);">下来选择框</font>_<font style="color:rgb(44, 62, 80);">; 数据表名称或数据表ID</font>

**<font style="color:rgb(44, 62, 80);">数据表:</font>**_<font style="color:rgb(44, 62, 80);">字符串</font>_<font style="color:rgb(44, 62, 80);">; 输入数据表的名称或ID</font>

**<font style="color:rgb(44, 62, 80);">记录详情:</font>**_<font style="color:rgb(44, 62, 80);">字典</font>_<font style="color:rgb(44, 62, 80);">; 输入记录的详情, 如, </font>`<font style="color:rgb(44, 62, 80);">{"序号": 1, "备注内容": "这是一条备注"}</font>`<font style="color:rgb(44, 62, 80);"> , 支持的字段格式如下, 复杂示例, </font>`<font style="color:rgb(44, 62, 80);">{"</font><font style="color:rgba(0, 0, 0, 0.85);">多行文本</font><font style="color:rgb(44, 62, 80);">": "string", "数字": 111, "单选": "选项1", "多选": ["选项1", "选项2"], "日期": "2021-12-12", "复选框": True, "附件": ["路径1", "路径2"]}</font>`<font style="color:rgb(44, 62, 80);">, 其他格式字段后续看业务需求提供支持</font>

| **<font style="color:rgba(0, 0, 0, 0.85);">字段类型</font>** | **<font style="color:rgba(0, 0, 0, 0.85);">value结构</font>** | **<font style="color:rgba(0, 0, 0, 0.85);">描述</font>** |
| --- | --- | --- |
| <font style="color:rgba(0, 0, 0, 0.85);">多行文本</font> | <font style="color:rgba(0, 0, 0, 0.85);">string</font> | <font style="color:rgba(0, 0, 0, 0.85);">值</font> |
| <font style="color:rgba(0, 0, 0, 0.85);">数字</font> | <font style="color:rgba(0, 0, 0, 0.85);">写入时为number，列出时为string</font> | <font style="color:rgba(0, 0, 0, 0.85);">值</font> |
| <font style="color:rgba(0, 0, 0, 0.85);">单选</font> | <font style="color:rgba(0, 0, 0, 0.85);">string</font> | <font style="color:rgba(0, 0, 0, 0.85);">选项名称</font> |
| <font style="color:rgba(0, 0, 0, 0.85);">多选</font> | <font style="color:rgba(0, 0, 0, 0.85);">array</font> | <font style="color:rgba(0, 0, 0, 0.85);">包含多个选项名字符串的数组</font> |

| 附件 | list[str] | 路径的列表 |


### <font style="color:rgb(44, 62, 80);">指令输出</font>
**<font style="color:rgb(44, 62, 80);">记录ID</font>**<font style="color:rgb(44, 62, 80);"> </font>_<font style="color:rgb(44, 62, 80);">字符串</font>_<font style="color:rgb(44, 62, 80);">; 返回创建记录的ID</font>
