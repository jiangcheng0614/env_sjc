# 描述
选中word中段得范围的范围

![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1715220960185-8cd69c52-949e-4f58-ae57-3410be132e5e.png)

## 参数配置
### 指令输入
#### 常规
word对象： 文档对象

起始段位置:  选择起始段的位置

结束段位置:  选择结束段的位置

###  指令输出
**选中范围: **选择范围的内容，并且在word中选择该内容
