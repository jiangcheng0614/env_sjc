# 描述
该指令实现网页中的日期选择





# 参数配置
## 指令输入
### 常规
**web_page**：网页对象，操作元素所在网页对象

**日期框**：web元素，包含1-31号的日期大框

**目标日**：字符串|数字，年月日中的日





### 高级
**点击方式**：下拉框

+ 单击
+ 双击





## 指令输出
无
