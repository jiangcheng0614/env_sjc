# 描述
获取博主主页笔记案例详细信息，适用于合作的博主进行数据关注场景，

阅读，点赞，收藏，标题，日期，链接

											![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1693226035170-2dd87ee3-cf7e-46bc-aa3d-9871dd857b04.png)

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1693224202482-9d8de391-d397-4205-b9b9-8a94beec78f1.png)



# **输入**
### **web_page：**
任意一个已登录小红书后台的网页对象

### 博主主页已打开：
如果当前网页对象是博主主页（比如已经运行了【博主主页-数据概览&粉丝画像-合作博主】指令），则勾选

不勾选，会重新跳转到博主主页。

### 博主id：
可以在地址栏中看到

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1693225200269-26031b48-7cfe-4d93-bf3f-f29822bccb60.png)

### 流量类型：
下拉可选

全部流量

自然流量

### 笔记类型：
下拉可选

全部类型

合作笔记

图文笔记

视频笔记



# 输出
### invoke_result：
二维列表，第一项是列名
