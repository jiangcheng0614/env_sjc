# 描述
获取博主主页流量占比详细数据，适用于合作的博主进行数据关注场景，

曝光_博主个人主页，曝光_关注页，曝光_发现页，曝光_附近页，曝光_其他，曝光_搜索页，阅读_博主个人主页，阅读_关注页，阅读_发现页，阅读_附近页，阅读_其他，阅读_搜索页

# **输入**
### **web_page：**
任意一个已登录小红书后台的网页对象

### 博主主页已打开：
如果当前网页对象是博主主页（比如已经运行了【博主主页-数据概览&粉丝画像-合作博主】指令），则勾选

不勾选，会重新跳转到博主主页。

### 博主id：
可以在地址栏中看到

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1693225200269-26031b48-7cfe-4d93-bf3f-f29822bccb60.png)

### 笔记类型：
下拉可选

日常笔记

合作笔记

### 流量类型：
下拉可选

全流量

仅自然流量

### 时间段：
下拉可选

近30日

近90日



# 输出
### invoke_result：
二维列表，第一项是列名
