# 描述
获取博主指标信息

博主, 博主id_, 内容标记, 特征标记, 图文价, 视频价, 粉丝数, 粉丝量变化幅度, 活跃粉丝占比, 互动粉丝占比, 阅读中位数, 互动中位数, 百赞笔记比例, 图文阅读中位数, 图文互动中位数, 预估阅读单价图文, 千赞笔记比例图文, 百赞笔记比例图文, 视频阅读中位数, 视频互动中位数, 视频完播率, 预估阅读单价视频, 千赞笔记比例视频, 百赞笔记比例视频

# **输入**
### **web_page：**
任意一个已登录小红书后台的网页对象

### 内容类目：
严格按照页面上【内容类目】的文本填入，只支持一级类目

### 抓取页数：
自定义要抓取的总页数（页数过大可能会有风控，请自行确认）

### 排序：
对数据搜索结果的排序，可下拉选择：

智能排序（网站默认）

按粉丝数

按阅读量

按互动数



# 输出
### invoke_result：
二维列表，第一项是列名
