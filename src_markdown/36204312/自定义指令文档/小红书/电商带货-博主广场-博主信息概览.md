# 描述
获取博主指标信息

博主，博主id，标签，内容标记，粉丝数，活跃粉丝占比，笔记互动中位数，直播观看人数，近期带货场次，场均销售额区间

# **输入**
### **web_page：**
任意一个已登录小红书后台的网页对象

### 内容类目：
严格按照页面上【内容类目】的文本填入，只支持一级类目

### 抓取页数：
自定义要抓取的总页数（页数过大可能会有风控，请自行确认）

### 排序：
对数据搜索结果的排序，可下拉选择：

默认排序

粉丝排序

活跃粉丝排序

互动排序

直播观看排序

带货场次排序

销售额排序

# 输出
### invoke_result：
二维列表，第一项是列名
