# 指令描述
该指令实现生意参谋服务板块的绩效相关的子模块的时间选择功能 , 支持快捷日期选择.

当前可以使该指令实现时间选择的部分如图所示 :

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677145243699-ebe55e10-e717-4bf8-a32d-5ee7e9dfa84c.png)

支持模块 : 服务-店铺绩效 / 服务-客服绩效 / 服务-绩效明细

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677145315030-438337cb-5c57-44cf-96c5-353d94b2c63d.png)

**输入参数 : **

日期 : 输入具体的日期字符串  例 : 2023-02-01.  也可以输入当前页面支持的快捷日期 例: 日/月/近30天..

**输出参数 : **

页面统计时间 : 返回当前页面显示的数据统计时间 .
