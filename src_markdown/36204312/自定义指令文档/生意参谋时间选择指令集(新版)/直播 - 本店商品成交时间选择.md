# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677152379755-f38314f9-2e1c-47e2-9d77-af74f095221c.png)

**输入参数 : **

日期 : 输入具体的日期字符串  例 : 2023-02-01.  只支持单日.

**输出参数 : **

页面统计时间 : 返回当前页面显示的数据统计时间 .
