# 指令描述
该指令实现生意参谋服务板块的监控相关的子模块的时间选择功能 , 支持快捷日期选择.

当前可以使该指令实现时间选择的部分如图所示 :

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677151971972-a709505b-6cd0-4a78-a366-ec676f936bbb.png)

支持模块 : 服务-核心监控

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677151992718-53fc7a99-8643-49a4-8c20-ee94c4b3c2de.png)

**输入参数 : **

日期 : 输入具体的日期字符串  例 : 2023-02-01.  也可以输入当前页面支持的快捷日期 例: 日/月/近30天..

**输出参数 : **

页面统计时间 : 返回当前页面显示的数据统计时间 .
