# 指令描述
该指令实现生意参谋一些特定页面的时间选择功能 , 标题URL指 通过网址拼接时间信息进行跳转目标页面 . 

目前生意参谋可以使用URL跳转来实现时间选择的模块有 : 流量 / 内容 / 首页 / 市场 / 竞争  这些板块

可以使用改指令选择时间的网址 截图示例 : 

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677143311752-465ea90d-875f-4216-8a96-e11cbd4be6ea.png)

<font style="color:#DF2A3F;">注 : 类似网址后面带有类似 dateRange=2023-02-23%7C2023-02-23&dateType=today 的信息就表明可以使用URL跳转选择时间. </font>

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677143441026-243d5416-2ce3-4e86-99b4-f46a766fc60e.png)

**输入参数 : **

日期 : 输入具体的日期字符串  例 : 2023-02-01. 目前只支持单日时间跳转, 期待后续更多跳转时间开发

web_page : 当前需要进行时间选择网页的网页对象 . 



# 适用的浏览器类型
1. Google Chrome浏览器
2. 影刀浏览器
3. Edge浏览器
4. 360浏览器
