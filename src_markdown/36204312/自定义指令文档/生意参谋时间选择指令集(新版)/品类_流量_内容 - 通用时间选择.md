# 指令描述
该指令实现生意参谋 _**品类/流量/内容 **_板块的时间选择功能 , 支持快捷日期选择.

当前可以使该指令实现时间选择的部分如图所示 :

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677152193567-0f733c0b-efa5-4b5c-934d-63e6dfa4bf07.png)



# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677152211133-92d09b82-c417-4f16-9efa-586e883d4fcb.png)

**输入参数 : **

日期 : 输入具体的日期字符串  例 : 2023-02-01.  也可以输入当前页面支持的快捷日期 例: 日/月/近30天..

**输出参数 : **

页面统计时间 : 返回当前页面显示的数据统计时间 .
