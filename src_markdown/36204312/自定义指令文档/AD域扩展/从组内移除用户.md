# 描述
将指定用户从组内移除

![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717071557970-a477881a-ac61-416d-b799-29f8361db821.png)**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**grouo_cn**：组CN

**user_cn**：用户CN

## 指令输出
**无**
