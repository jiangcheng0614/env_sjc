# 描述
搜索指定组内的用户信息

![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717071871174-1a221906-6756-4250-8e89-a2e832680ddd.png)**LDAP连接对象：与指定AD域连接的LDAP连接对象**

**AD域名**：域名称

**grouo_cn**：组CN

**members_list：组内成员的信息列表**
