# 描述
往连接成功的AD域里删除一个组织

# ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717067322937-de6a17f0-6b83-4824-b030-cd5746729c86.png)
## 指令输入
### 常规
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**父组织位置**：新建组织所在位置

**要增加的组织名称**：新建的组织名

## 指令输出
**无**
