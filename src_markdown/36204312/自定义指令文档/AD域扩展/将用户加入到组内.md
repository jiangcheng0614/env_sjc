# 描述
将指定用户添加到组内

### ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717071299230-643335c2-7afd-4b82-a8eb-4f6befb0fdd9.png)
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**grouo_cn**：组CN

**user_cn**：用户CN

## 指令输出
**无**
