# 描述
往指定的组织里增加一个用户

![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717067553660-0a3f2f1f-5ada-4f5d-8440-be4f92fce1ae.png)

## 指令输入
### 常规
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**父组织位置**：新建组织所在位置

**显示名称**：新建用户显示的名称

**域登录名**：用户域的登录名称

**用户登录名**：用户用来登录的名称

**CN**：用户CN

## 指令输出
**无**
