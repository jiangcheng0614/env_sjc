# 参数配置
## 指令输入
### 常规
**IP地址**：AD域服务器所在的IP地址

**用户名称**：登录服务器界面所需的用户名称

**用户密码**：登录服务器界面所需的用户密码

## 指令输出
**LDAP连接对象：一个连接对象**
