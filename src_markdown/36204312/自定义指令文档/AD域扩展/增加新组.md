# 描述
增加一个新的组

## ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717070712599-41e3a486-69c0-4994-b17f-a2f6137d8d40.png)指令输入
### 常规
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**父组织位置**：新建组织所在位置

**CN**：组CN

## 指令输出
**无**
