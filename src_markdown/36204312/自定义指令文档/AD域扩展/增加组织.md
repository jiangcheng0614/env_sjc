# 描述
往连接成功的AD域里新建一个组织

# ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717062756935-c867ec23-8fb1-4eb5-9309-2b0856467bff.png)参数配置
## 指令输入
### 常规
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**父组织位置**：新建组织所在位置

**要增加的组织名称**：新建的组织名

## 指令输出
**无**
