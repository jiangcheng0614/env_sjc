# 扩展：
### 常见LDAP用户属性
1. **身份信息**：
    - cn：Common Name（常用名称）
    - sn：Surname（姓氏）
    - givenName：Given Name（名字）
    - uid：User ID（用户ID）
    - displayName：显示名称
    - distinguishedName：DN（区别名）
2. **联系信息**：
    - mail：电子邮件地址
    - telephoneNumber：电话号码
    - homePhone：家庭电话
    - mobile：移动电话
    - postalAddress：邮政地址
    - l：城市
    - st：州/省
    - postalCode：邮政编码
    - c：国家
3. **组织信息**：
    - title：职位
    - department：部门
    - company：公司
    - manager：经理
    - memberOf：所属组
4. **账户信息**：
    - userPrincipalName：用户主体名称
    - sAMAccountName：SAM账户名
    - objectGUID：对象GUID
    - objectSid：对象SID
    - userAccountControl：账户控制属性
    - pwdLastSet：最后设置密码时间
    - lastLogon：最后登录时间
    - accountExpires：账户过期时间
5. **地址信息**：
    - streetAddress：街道地址
    - postOfficeBox：邮政信箱
    - physicalDeliveryOfficeName：办公地点名称
