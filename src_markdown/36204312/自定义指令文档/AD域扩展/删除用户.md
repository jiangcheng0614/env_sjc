# 描述
删除指定用户CN的用户

## ![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1717069133455-85d07358-80c3-4403-9569-905cfe6c71a5.png)指令输入
### 常规
**LDAP连接对象**：与指定AD域连接的LDAP连接对象

**AD域名**：域名称

**父组织位置**：新建组织所在位置

**CN**：用户的CN

## 指令输出
**无**
