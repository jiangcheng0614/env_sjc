# 一.功能描述
关闭word文档，可以选择是否关闭word进程。

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1677574743506-12c9dec9-0f22-4e0b-938e-c14ce6025aae.jpeg)



# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**<font style="color:rgb(51, 51, 51);">word对象：</font>**

此参数需用扩展指令集内【打开word文档】控件的返回值来做为入参。

_<font style="color:rgb(102, 102, 102);">注：</font>_<font style="color:#585A5A;">【</font>_<font style="color:#585A5A;">打开word文档】是否可见参数需选择”是”</font>__<font style="color:rgb(102, 102, 102);">。</font>_

**关闭word程序：**

默认选择“否”，选择“是”关闭指定的文档退出word。选择“否”，只关闭word，不退出word进程。

### <font style="color:rgb(51, 51, 51);">输出</font>
**<font style="color:rgb(51, 51, 51);">返回值：</font>**无

例：如下图，退出word执行成功后，word进程也会关闭。
