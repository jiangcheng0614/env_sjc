# 描述
由顺丰快递发货的订单，根据订单号对应的运单号完成快递退回。															

![](https://cdn.nlark.com/yuque/0/2023/png/35870027/1693398908197-74d8a448-1ded-4a73-8744-956628ba3de6.png)

# **输入**
### **运单号：**
多个单号以英文逗号隔开

### 输出
### invoke_result：
字符串，全部退回返回全退；部分退回返回退回失败的运单号（多个运单号以逗号隔开）
