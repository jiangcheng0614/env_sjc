# 描述
该指令实现了在企业微信查找联系人或群聊并发送消息的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1715843441765-baf5b1b2-5316-415d-b4f8-72e16a1d3ae1.png)

# 参数配置
## 指令输入
### 常规
**名称**：字符串，联系人名称或者群聊名称



**查找规则**：下拉框

+ 等于
+ 包含



**查找类型**：下拉框

+ 联系人
+ 群聊



**发送内容**：字符串，需要发送的消息内容





### 高级
****



## 指令输出
无
