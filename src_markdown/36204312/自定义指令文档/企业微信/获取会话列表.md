# 描述
该指令实现了在企业微信根据会话标签获取会话列表的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1726037032939-3b3c3c74-e639-42d4-8762-39ab509d665a.png)

# 参数配置
## 指令输入
### 常规
**会话标签**：下拉框

+ 未读
+ @我
+ 单聊
+ 群聊
+ 内部聊天
+ 外部聊天
+ 标记

**滚动次数**: 整数, 每次滚动鼠标的次数,用于滚动会话列表. 请根据实际情况调整大小,避免滚动太多导致抓漏的情况

### 高级
**无**



## 指令输出
**会话列表:** 列表, 抓取到的会话名称列表

# 注意事项
请确保左侧导航栏有 **分组 **按钮, 如果没有,可以尝试升级企业微信
