# 描述
在元素选择器列表中，等待任意一组元素的出现

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1690715606019-f30595f5-57b0-41b4-9641-c5118a46b493.png)



# 参数配置
## 指令输入
### 常规
**网页对象：**网页对象

**元素1：**元素1

**元素2：**元素2

**元素3：**元素3，非必填

**元素4：**元素4，非必填

**元素5：**元素5，非必填

**超时时间(S)：**等待元素出现的超时时间，单位：秒，如果填-1则一直等待到出现



### 高级
**元素选择器列表：**通过【组合多元素(web)】指令，获取元素选择器列表，非必填



## 指令输出
**相似元素名称：**出现的相似元素的选择器名称

**相似元素列表：**出现的相似元素列表
