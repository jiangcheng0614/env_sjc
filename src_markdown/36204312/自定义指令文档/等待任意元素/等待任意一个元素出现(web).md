# 描述
在一组选择器中，等待任意一个元素的出现

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1688433157378-850cb04f-8f06-4fec-a0aa-1365e4e12192.png)



# 参数配置
## 指令输入
### 常规
**网页对象：**网页对象

**元素1：**元素1

**元素2：**元素2

**元素3：**元素3，非必填

**元素4：**元素4，非必填

**元素5：**元素5，非必填

**超时时间(S)：**等待元素出现的超时时间，单位：秒



### 高级
**元素选择器列表：**通过【组合多元素(web)】指令，获取元素选择器列表，非必填



## 指令输出
**网页元素名称：**出现的元素的选择器名称

**网页元素：**出现的网页元素对象
