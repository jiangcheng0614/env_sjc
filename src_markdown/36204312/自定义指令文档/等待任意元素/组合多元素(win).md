# 描述
将多个元素选择器组合成列表

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1688350273141-ca1e88bb-2aaa-47ed-b1e2-2c49713178a8.png)



# 参数配置
## 指令输入
### 常规
**元素1：**元素1

**元素2：**元素2

**元素3：**元素3

**元素4：**元素4

**元素5：**元素5





## 指令输出
**元素选择器列表：**多个元素选择器组成的列表，会自动过滤空值



# 常见问题
## 1.不需要用到5个元素
可以根据实际需求传递2-4个，会过滤空值



## 2.超过5个元素
可以用重复使用当前指令，然后把两个输出的列表合并成一个列表
