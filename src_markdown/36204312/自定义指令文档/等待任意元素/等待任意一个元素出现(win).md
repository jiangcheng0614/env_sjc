# 描述
在一组选择器中，等待任意一个元素的出现

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1688433285469-e4f53aa3-1790-4d1a-bef6-78d80060a580.png)



# 参数配置
## 指令输入
### 常规
**元素1：**元素1

**元素2：**元素2

**元素3：**元素3，非必填

**元素4：**元素4，非必填

**元素5：**元素5，非必填

**超时时间(S)：**等待元素出现的超时时间，单位：秒



### 高级
**元素选择器列表：**通过【组合多元素(win)】指令，获取元素选择器列表，非必填





## 指令输出
**win元素名称：**出现的元素的选择器名称

**win元素对象：**出现的win元素对象
