# 描述
该指令实现了获取元素背景图片的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1713840879993-225b1041-dca2-4a35-bce4-e1a5291ec83f.png)

# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，操作元素所在网页对象

**操作目标**：web元素，需要获取背景图片的元素



### 高级
无



## 指令输出
**背景图片**：字符串，具体形式根据网页决定，有的是base64,有的是url





# 注意事项
一定要选对元素，选父级或者子级可能会获取不到对应的背景图
