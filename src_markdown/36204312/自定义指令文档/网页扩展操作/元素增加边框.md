# 描述
该指令实现了给元素增加边框的功能

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1721719117558-2b34b419-dde9-4065-a64c-c1b0bf9cb16d.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，操作元素所在网页对象

**操作目标**：web元素，需要增加边框的元素

**粗细**：整数，边框的大小

**样式**：下拉框, 边框的样式

+ 直线
+ 虚线

**颜色**：字符串，边框的颜色



### 高级
无



## 指令输出
无
