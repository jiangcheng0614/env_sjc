# 描述
该指令实现了根据已有网页对象的浏览器类型获取当前激活的网页对象，用于新增网页对象的场景

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1710295960212-31e8fb08-7fe5-4010-a0ac-34d0d8f76451.png)



# 参数配置
## 指令输入
### 常规
网页对象：网页对象，已有的网页对象





### 高级




## 指令输出
web_page：网页对象，当前激活的网页对象





# 注意事项


# 使用场景
操作网页时，如果新增网页对象。用原生的“获取当前激活网页对象”指令，会写死浏览器类型，会导致后边不方便维护，使用当前指令根据已有网页对象的浏览器类型获取当前激活网页对象，就可以避免这个问题
