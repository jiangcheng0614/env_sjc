# 描述
该指令实现了获取元素字体颜色操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1710295903938-89c47739-99e2-4b9c-a634-ac1dd2ce6be1.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，操作元素所在网页对象

**操作目标**：web元素，需要获取字体颜色的元素



### 高级




## 指令输出
**字体颜色**：字符串，rgb的形式 -> rgb(xxx, xxx, xxx)





# 注意事项
一定要选对元素，选父级或者子级可能会获取不到对应的颜色
