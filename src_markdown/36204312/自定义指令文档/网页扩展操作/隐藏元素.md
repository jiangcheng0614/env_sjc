# 描述
该指令实现了将网页中的元素隐藏

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1710295840784-4b42d07a-a112-4724-a760-76041c3c4838.png)



# 参数配置
## 指令输入
### 常规
网页对象：网页对象，操作元素所在网页对象

操作目标：web元素，需要隐藏的元素



### 高级
无



## 指令输出
无



# 使用场景
1. 对于一些元素在可视区域，但是被顶部或者底部的标题栏遮挡导致点击不到的情况

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1703143540054-23acdac6-8512-4452-9584-f1844ee6d451.png)



如上图所示，顶部的标题栏会刚好遮住在那部分的元素，我们可以将顶部的标题览隐藏。需要操作的时候再显示出来
