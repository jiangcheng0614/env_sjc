# 描述
该指令实现了配置浏览器启动命令行的操作，目前针对谷歌浏览器

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1710295989863-de9bab26-f71b-4488-a3df-ccbe3f343f2f.png)



# 参数配置
## 指令输入
### 常规
**指定端口**：数字|字符串，选填，打开调试端口

**用户数据**：字符串，选填，指定谷歌用户数据存放文件夹，一般用于谷歌多开的场景

**指定用户**：字符串，选择指定的用户打开浏览器。

切换到指定用户后，在浏览器输入 chrome://version ,个人资料路径的最后一项就是指定用户的标识

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1709520805251-3db58593-e517-479a-a0ed-db1303b1b838.png)

**设置UA**：字符串，设置浏览器的 userAgent

按F12，打开浏览器控制台，输入 navigator.userAgent ，就可以拿到浏览器当前的UA



![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1709520962089-8d0d8301-2922-477b-b9ed-f3a180d1e479.png)



按图中操作后，再重复上面的操作，可以拿到不同的UA，模拟手机

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1709521211882-b6f10d87-9f24-455f-a54c-71df71881165.png)



**禁用图片**：复选框，禁用网页中图片显示，可以加快网页加载

**最大化**：复选框，浏览器窗口最大化

**无痕模式**：复选框，打开无痕模式

**隐藏崩溃弹窗**：复选框，隐藏崩溃弹窗





### 高级
无



## 指令输出
命令行：字符串，用于“打开网页”指令高级页传入
