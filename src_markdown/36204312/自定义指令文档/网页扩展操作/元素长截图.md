# 描述
该指令实现了对元素长截图的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1716772964153-20e83b63-d5e3-4b9b-aee6-07b0cc34d218.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，操作元素所在网页对象

**操作目标**：web元素，需要截图的元素

**保存路径**：字符串，保存截图的路径，png后缀的文件路径



### 高级
无



## 指令输出
无



## 注意事项
长截图是截取元素太长没有全部显示在网页中的元素，如果是未加载出来的元素是没法截取完整的
