# 描述
该指令实现了智能识别日期控件，然后选择指定日期的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1712468404803-193aa9da-bf89-4a26-bab1-8388889f35dd.png)

# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，填写所在网页对象

**日期框**：web元素，日期框要包含整个日期的年月日，以及向前向后切换按钮，可以查看下方示例

**开始日期**：字符串，格式：yyyy-mm-dd

**结束日期**：字符串，非必填，格式：yyyy-mm-dd





### 高级
无



## 指令输出
无



# 注意事项
并不是所有日期都能使用此指令，如果你遇到处理不了的日期框

1. 可以暂时使用之前的日期选择
2. 提供环境给我，我会尽量兼容进去
