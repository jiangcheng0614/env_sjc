# 描述
该指令实现了触发下拉框点击目标文本的功能

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1721719350539-09e433e0-30d1-4841-880e-e939027527c9.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，操作元素所在网页对象

**触发元素**：网页元素，点击触发下拉框的元素

**点击文本**：字符串，下拉框中需要点击的文本

**是否点击**：布尔值, 是否要点击文本



### 高级
无



## 指令输出
**网页元素**: 网页元素, 目标文本对应的网页元素
