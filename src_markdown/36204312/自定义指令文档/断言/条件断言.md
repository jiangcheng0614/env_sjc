# 描述
判断对象1和对象2的关系，判断错误则抛出异常

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1691027065129-9c96fb3b-ade2-45ef-9059-1f6ae8f29a9a.png)



# 参数配置
## 指令输入
### 常规
**对象1：**需要判断的对象

**关系：**对象1与对象2直接的关系

**对象2：**需要判断的对象

**错误信息：**判断错误抛出的异常信息





## 指令输出
无





# 使用场景
## 1.为了保证流程能够正常运行，对一些参数的校验
如果有多个参数需要校验，需要些大量的if判断，而使用此指令则可以减少一些工作量。流程看起来也简洁一些
