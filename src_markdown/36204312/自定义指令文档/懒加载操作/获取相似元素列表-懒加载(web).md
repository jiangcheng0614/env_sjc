# 描述
获取网页中懒加载相似元素列表，此列表的长度会动态变化；

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1684481393217-b8378abd-949c-4e7a-b085-185b024df916.png)





# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象

**操作目标：**懒加载相似元素组



****

### 高级
**等待元素存在(s)**：等待新元素存在的时间，如果填写-1则一直等待

**每次获取数量：**每次获取新元素的数量，如果填写-1则有多少新元素获取多少。如果你的场景在下滑的时候会刷新已经出

现的元素，那这个值越小越好，当然，这样会牺牲速度换稳定。

## 指令输出
**保存元素列表至：**获取到的懒加载相似元素列表

# 常见问题
## 1.<font style="color:rgb(31, 38, 51);">未找到指定ID的元素</font>
用相似元素列表指令也会遇到的问题。原因是影刀在捕获元素的时候，会给元素打一个uia-uid属性，而我们在操作元素的时候，网页发生了跳转动作或者重新加载，这个属性就会消失，导致出现的异常。



**解决方案：**将元素的跳转链接改成新标签页打开，需要注意的是，该元素必须是导致页面重新加载的元素



![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1684483816469-03539d2e-ef88-4aec-b06b-f59cbc33d41b.png)





## 2.元素很多
如果元素很多，并且不需要获取那么多元素，可以通过当前循环项的下标来判断是否停止操作

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1684483474241-842a9206-6fe3-4768-b0a3-c78f02a10468.png)
