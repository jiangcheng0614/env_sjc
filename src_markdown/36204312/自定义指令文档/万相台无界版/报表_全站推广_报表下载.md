# **输入**
### **网页对象：**
任意一个已登录万相台无界版后台的网页对象

### 归因周期：
选择归因周期

### 汇总周期_快捷日期选择：
下拉选择

快捷选择日期 昨日 过去七天等（选择快捷日期不用填写自定义日期）

### 汇总周期_自定义日期选择：
yyyy-mm-dd~yyyy-mm-dd

### 保存路径：
下载文件保存的文件夹路径，不需要带文件名

### 保存文件名：
下载文件保存的文件名

### 时间粒度：
汇总、分时（仅近30天）、分天、分周、分月

### 维度_是否全选：
下拉选择

是 否

# 输出
### 下载的文件列表：
一维列表，报表的完成路径
