# **输入**
### **web_page：**
任意一个已登录万相台无界版后台的网页对象

### 开始时间、结束时间：
YYYY-MM-DD

### 保存路径：
下载文件保存的文件夹路径，不需要带文件名

### 时间粒度：
下拉选择

汇总、分时（仅近30天）、分天、分周、分月

### 归因模型：
下拉选择

末次点击归因

### 归因周期：
下拉选择

1天累计数据、3天累计数据、7天累计数据、15天累计数据、30天累计数据

### 快捷日期：
下拉选择

无、昨日、上周、本月、上月、过去7天、过去15天、过去30天

### 维度：
下拉选择

单元、全选、时间、计划

# 输出
### 下载的文件列表：
一维列表，报表的完成路径
