# **输入**
### **web_page：**
任意一个已登录万相台无界版后台的网页对象

### 开始时间、结束时间：
YYYY-MM-DD

### 保存路径：
下载文件保存的文件夹路径，不需要带文件名

### 版本：
下拉选择

账户总体、持续投放、套餐包投放、快速起量、管家版、其他

### 场景：
下拉选择

全部解决方案、直播间成长、商品打爆、直播间人群运营

### 转化效果周期：
下拉选择

转化效果周期 - 1天、转化效果周期 - 3天、转化效果周期 - 7天、转化效果周期 - 15天、转化效果周期 - 30天

### 快捷日期：
下拉选择

无、昨日、上周、本月、上月、过去7天、过去15天、过去30天

### 类型：
下拉选择

推广、定向、创意

# 输出
### 下载的文件列表：
一维列表，报表的完成路径
