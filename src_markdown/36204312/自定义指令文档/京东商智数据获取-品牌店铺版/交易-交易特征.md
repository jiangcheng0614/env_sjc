# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676022877304-fe4e0265-c7f0-474c-a0c1-a7db1f92be57.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

**类目特征：**复选框

**品牌特征特征：**复选框

**商品价格带特征：**复选框

**新老客户特征：**复选框

**支付方式特征：**复选框

**客单件数特征：**复选框

**渠道类征：**复选框

## 输出
**文件存储列表：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
