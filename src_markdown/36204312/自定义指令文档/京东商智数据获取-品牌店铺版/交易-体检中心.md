# 描述
获取京东商智品牌版中**交易-体检中心**页面下的**体检中心、店铺流量、店铺交易、页面装修**数据，并将其写入Excel保存到指定位置。

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676279234712-0bdf9b9c-521a-4dfb-9022-3dee7c1bcfc0.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

## 输出
**文件存储列表：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
