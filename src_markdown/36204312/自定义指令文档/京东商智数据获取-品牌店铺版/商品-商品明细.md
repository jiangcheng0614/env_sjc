# 描述
获取京东商智品牌版中**商品-商品明细**页面数据，并将其写入Excel保存到指定位置。

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676013949014-0e3577ec-e136-4b4d-9853-ec186937024d.png)



# 配置说明![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676015048496-8e9150a8-38cc-4000-9aee-81c6a2369e8c.png)
## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

**渠道****<font style="color:#000000;">：</font>**数据类型为字符串，默认为'全部渠道'

## 输出
**文件存储列表：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
