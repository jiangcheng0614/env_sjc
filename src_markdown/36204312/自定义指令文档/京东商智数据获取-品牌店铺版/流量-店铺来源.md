# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1675827918618-f90b8465-1ee1-4797-9d25-f085a78d5ae3.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

**<font style="color:#000000;">数据对比：</font>**数据类型为字符串，选择 '环比' 或者 '同比', 默认为'环比'

**渠道****<font style="color:#000000;">：</font>**数据类型为字符串，默认为'全部'

## 输出
**文件路径：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
