# 描述
获取京东商智品牌版中**商品-商品概况**页面下的**商品概览、商品榜单**数据，并将其写入Excel保存到指定位置。

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676379246646-ae3350fb-4831-4fbd-941b-ddc826fc4ecd.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

**数据对比：**数据类型为字符串，选择'环比'或者'同比'，默认为'环比'

**渠道****<font style="color:#000000;">：</font>**数据类型为字符串，默认为'全部'

## 输出
**文件存储列表：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
