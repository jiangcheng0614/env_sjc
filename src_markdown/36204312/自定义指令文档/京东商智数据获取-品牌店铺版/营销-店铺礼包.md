# 描述
获取京东商智品牌版中**营销-店铺礼包**页面下的**活动效果**数据，并将其写入Excel保存到指定位置。

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1676378577355-8182ecfb-c67d-46eb-a585-8f2e2106a863.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

## 输出
**文件路径：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
