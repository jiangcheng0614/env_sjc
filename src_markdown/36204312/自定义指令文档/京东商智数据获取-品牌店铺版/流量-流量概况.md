# 描述
获取京东商智品牌版中**流量-流量概况**页面下的**核心指标、流量趋势**数据，并将其写入Excel保存到指定位置。

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1675771654516-566a4662-40d7-4f76-a4db-8e5dae040352.png)



# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1675822299639-741bf1db-a9d5-4796-9f88-184f3be43f82.png)

## 输入
**数据保存路径:** 数据类型为路径字符串，请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据时间：</font>**数据类型为日期格式字符串，时间格式为 '2022-01-02', 默认为昨日

**<font style="color:#000000;">核心指标_数据对比：</font>**数据类型为字符串，选择 '环比' 或者 '同比', 默认为'环比'

**流量趋势_时间颗粒****<font style="color:#000000;">：</font>**数据类型为字符串，选择 '日' 或者 '小时', 默认为'日'

## 输出
**文件路径：**数据类型为列表，输出指令获取的数据保存在本地的文件路径列表
