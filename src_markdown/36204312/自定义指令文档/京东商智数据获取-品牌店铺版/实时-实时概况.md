# 描述
获取京东商智品牌版中实时-实时概况数据。

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1675671562396-c38d15d2-d60c-4684-920c-8a235ee8630b.png)



# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1675672428249-c2e2c174-d3ca-428f-be13-23237a5f7136.png)

## 输入
**数据保存路径:** 请输入下载文件需要保存的文件夹路径

**<font style="color:#000000;">数据下载日：</font>**下载日期, 仅支持传参 今日 或者 昨日

**<font style="color:#000000;">数据对比日：</font>**日期格式：2022-01-02 或者为：去年同期  两种可供选择，默认为昨日日期



## 输出
**文件路径：**输出指令获取的数据保存在本地的文件路径
