# 一.功能描述
用于抖店-多店铺的店铺切换。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1682212439575-dac1b8f0-019f-4d2f-a28b-c5dba3b99c1c.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1682212583868-c64ee06f-352f-4c01-a450-4a0401fde8ac.png)

## 1.输入参数
**店铺名称：**  
输入需要切换到的店铺名称或者店铺的位置从1开始。

*目前只支持谷歌浏览器。

## 2.输出参数
**所有店铺名称：**  
以列表的形式，返回该账号下所有的店铺名称。
