# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/40912025/1714268849372-4949fcb9-ff5f-48f9-8800-933652efaae8.png)

### 指令输入
#### 常规
**开始日期触发器: 显示时间段选择的按钮**

**开始时间: 写入起始时间段  格式为yyyy-mm-dd-hh-mm-ss例如：(2024-3-28-04-12-45)**

**结束时间：写入结束时间段 格式为yyyy-mm-dd-hh-mm-ss例如：(2024-3-28-04-12-45)**
