# 描述
调用[翔云证件识别接口](http://www.netocr.com/products.html)，对图片进行OCR识别。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676944798979-08d7b705-2ea3-428f-a9db-f73c7f2c8453.png)



# **输入**
### **key：**
[翔云](http://www.netocr.com/user/index.html?sul=users_order)账号的OCR key

### secret：
翔云账号的OCR secret



![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676893020440-89f3590a-b091-4069-9714-bb289fec9625.png)

### 通行证类型：
下拉框可选项

+ <font style="color:rgb(50, 50, 50);">台湾居民往来大陆通行证 1992 版-照片页</font>
+ <font style="color:rgb(50, 50, 50);">港澳居民来往内地通行证-照片页</font>
+ <font style="color:rgb(50, 50, 50);">港澳居民来往内地通行证-机读码页</font>
+ <font style="color:rgb(50, 50, 50);">台湾居民来往大陆通行证 2015 版-照片页</font>
+ <font style="color:rgb(50, 50, 50);">中华人民共和国往来港澳通行证 2005 版</font>
+ <font style="color:rgb(50, 50, 50);">台湾居民往来大陆通行证 2015 版-机读码页</font>
+ <font style="color:rgb(50, 50, 50);">大陆居民往来台湾通行证 1992 版-照片页</font>
+ <font style="color:rgb(50, 50, 50);">往来台湾通行证2017版-照片页</font>
+ <font style="color:rgb(50, 50, 50);">往来港澳通行证 2014 版-照片页(卡式港澳通行证)</font>



### 图片：
证件图片的路径，字符串

若批量识别，请传入一维列表 ["path1", "path2", ···  ]

# 输出
### invoke_result：
**单个**图片识别的结果，返回**一维列表**，列表中的第一项是识别状态 ，内容都是**字典形式**

**批量**图片识别，一次性返回**二维列表**。

参考[错误码](http://www.netocr.com/apiCenter/index.html)
