# 描述
调用[翔云发票查验接口](http://www.netocr.com/verInvoiceFor2.html)，查验发票真实性。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676962094433-3c201a72-76ed-4a8c-94a9-82e4edd9690a.png)



# **输入**
### **key：**
[翔云](http://www.netocr.com/user/index.html?sul=users_order)账号的OCR key

### secret：
翔云账号的OCR secret



![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676893020440-89f3590a-b091-4069-9714-bb289fec9625.png)

### <font style="color:rgb(0, 0, 0);">根据纳税人查验的票种，其输入的字段也不相同，其中：</font>
<font style="color:rgb(0, 0, 0);">（1）</font>**<font style="color:rgb(0, 0, 0);">增值税专用发票、增值税电子专用发票</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和合计金额(不含税)；  
</font><font style="color:rgb(0, 0, 0);">（2）</font>**<font style="color:rgb(0, 0, 0);">增值税普通发票、增值税电子普通发票（含通行费发票）、增值税普通发票（卷票）</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和校验码后6位；  
</font><font style="color:rgb(0, 0, 0);">（3）</font>**<font style="color:rgb(0, 0, 0);">机动车销售统一发票</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和合计金额（不含税价）；  
</font><font style="color:rgb(0, 0, 0);">（4）</font>**<font style="color:rgb(0, 0, 0);">二手车销售统一发票发票</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和合计金额；  
</font><font style="color:rgb(0, 0, 0);">（5）</font>**<font style="color:rgb(0, 0, 0);">电子发票（增值税专用发票）、电子发票（普通发票）</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期、合计金额（含税）；  
</font><font style="color:rgb(0, 0, 0);">（6）电子发票服务平台开具的纸质</font>**<font style="color:rgb(0, 0, 0);">增值税专用发票</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和合计金额（不含税）；  
</font><font style="color:rgb(0, 0, 0);">（7）电子发票服务平台开具的纸质</font>**<font style="color:rgb(0, 0, 0);">增值税普票发票</font>**<font style="color:rgb(0, 0, 0);">：发票代码、发票号码、开票日期和校验码（密码区全电发票号码后6位）；  
</font><font style="color:rgb(0, 0, 0);">（8）</font>**<font style="color:rgb(0, 0, 0);">电子发票（铁路电子客票）</font>**<font style="color:rgb(0, 0, 0);">：发票号码、开票日期和合计金额；  
</font><font style="color:rgb(0, 0, 0);">（9）</font>**<font style="color:rgb(0, 0, 0);">电子发票（航空运输电子客票行程单）</font>**<font style="color:rgb(0, 0, 0);">：发票号码、开票日期、合计金额（含税）。</font>

<font style="color:rgb(82, 82, 82);">开票日期格式为YYYY-MM-DD</font>

<font style="color:rgb(82, 82, 82);">合计金额类型是字符串</font>

<font style="color:rgb(82, 82, 82);">当日开具发票当日可查验。</font>

<font style="color:rgb(82, 82, 82);">每份发票每天最多可查验5次。</font>

<font style="color:rgb(82, 82, 82);">可查验最近5年内增值税发票管理系统开具的发票。</font>

# 输出
### invoke_result：
返回**一维列表**，列表中的第一项是识别状态 ，内容都是字典形式**，状态**对应**成功**就是**查验通过。**

[错误码](http://www.netocr.com/apiCenter/index.html)参考
