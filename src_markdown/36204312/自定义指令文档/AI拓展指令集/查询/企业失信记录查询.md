# 描述
调用[翔云失信被执行人查询接口](http://www.netocr.com/break_faith.html)，查询全国范围内法人或组织被执行信息。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676967296128-f717434b-5275-4c7f-88c6-03f7b8a7a024.png)



# **输入**
### **key：**
[翔云](http://www.netocr.com/user/index.html?sul=users_order)账号的OCR key

### secret：
翔云账号的OCR secret



![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676893020440-89f3590a-b091-4069-9714-bb289fec9625.png)

### 名称：
必填，单个查询，输入字符串类型

若批量查询，请传入一维列表 ["name1", "name2", ···  ]

### 组织机构代码：
必填，单个查询，输入字符串类型

若批量查询，请传入一维列表 ["id1", "id2", ···  ]



若批量查询，名称与组织机构代码的列表长度要相等



# 输出
### invoke_result：
**单个**查询的结果，返回**一维列表**，列表中的第一项是识别状态 ，内容都是**字典形式**

**批量**查询结果，一次性返回**二维列表**。

[错误码](http://www.netocr.com/apiCenter/index.html)参考
