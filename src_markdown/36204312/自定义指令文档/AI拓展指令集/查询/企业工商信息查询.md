# 描述
调用[翔云企业工商信息接口](http://www.netocr.com/verLicence.html)，通过企业（个体）名称查询企业（个体）信息。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676967788282-b1969d91-c241-451e-90ac-147bfe350391.png)



# **输入**
### **key：**
[翔云](http://www.netocr.com/user/index.html?sul=users_order)账号的OCR key

### secret：
翔云账号的OCR secret



![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676893020440-89f3590a-b091-4069-9714-bb289fec9625.png)

### AI版本：
下拉框可选项

+ <font style="color:rgb(50, 50, 50);">标准版</font>
+ <font style="color:rgb(50, 50, 50);">增强版</font>

### 企业名称：
单个查询，请输入字符串类型

若批量识别，请传入一维列表 ["name1", "name2", ···  ]

# 输出
### invoke_result：
**单个**查询的结果，返回**一维列表**，列表中的第一项是识别状态 ，内容都是**字典形式**

**批量**查询结果，一次性返回**二维列表**。

[错误码](http://www.netocr.com/apiCenter/index.html)参考
