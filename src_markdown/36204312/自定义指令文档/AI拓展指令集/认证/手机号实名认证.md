# 描述
调用[翔云手机号实名认证接口](http://www.netocr.com/verTelOrd.html)，核查上传姓名+身份证号+手机号是否与权威数据一致。<font style="color:rgb(255, 255, 255);">致</font>

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676979352203-ebf6240d-057f-49f8-baae-e40964970355.png)



# **输入**
### **key：**
[翔云](http://www.netocr.com/user/index.html?sul=users_order)账号的OCR key

### secret：
翔云账号的OCR secret



![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1676893020440-89f3590a-b091-4069-9714-bb289fec9625.png)

### AI版本：
下拉框可选项

+ <font style="color:rgb(50, 50, 50);">标准版</font>
+ <font style="color:rgb(50, 50, 50);">增强版</font>

### <font style="color:rgb(50, 50, 50);">姓名：</font>
字符串

若批量识别，请传入一维列表 ["name1", "name2", ···  ]

### 身份证号：
字符串

若批量识别，请传入一维列表 ["id1", "id2", ···  ]

### 手机号：
必填，字符串

若批量识别，请传入一维列表 ["tel1", "tel2", ···  ]



若要批量识别，姓名列名与身份证号列表以及手机号的列表长度要相等

# 输出
### invoke_result：
**单个**认证的结果，返回**一维列表**，列表中的第一项是识别状态 ，内容都是**字典形式**

**批量**认证的结果，一次性返回**二维列表**。

[错误码](http://www.netocr.com/apiCenter/index.html)参考
