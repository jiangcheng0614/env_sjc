# 描述
该指令实现了全程智能侦测并屏蔽弹窗的操作

# 使用示例
主流程第一行调用即可，一般一个应用调用一次即可。

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1702606160072-570a1c70-9452-49c3-a4f3-5e92e341e0eb.png)



# 注意事项
该指令只有在操作元素失败时，才会智能侦测并屏蔽弹窗，如果没有操作元素则不会屏蔽。

如果用的是自动化接口操作，成功执行了也不会屏蔽弹窗
