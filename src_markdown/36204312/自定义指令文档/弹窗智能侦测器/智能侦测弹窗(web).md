# 描述
该指令实现了全程智能侦测弹窗并执行指定流程的操作

![](https://cdn.nlark.com/yuque/0/2024/jpeg/22242030/1711623271827-cb8a786e-0824-4bea-9b20-a1b4c415eae4.jpeg)



# 参数配置
## 指令输入
**执行流程：**字符串，侦测到弹窗后，需要执行的流程名称

**执行流程后的操作：**下拉框，

+ 执行上个步骤：当执行完指定的流程后，继续执行上一个未触发事件的步骤
+ 执行下个步骤：当执行完指定的流程后，继续执行下一个步骤

# 注意事项
该指令只有在操作元素失败时，才会智能侦测弹窗，如果没有操作元素则不会侦测到

如果用的是自动化接口操作，成功执行了也不会侦测到弹窗
