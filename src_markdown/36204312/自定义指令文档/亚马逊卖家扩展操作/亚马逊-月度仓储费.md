# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称：**支持中文和英文简写，例：1.美国、2.US

**保存文件夹：**下载文件保存的文件夹

**保存文件名：**下载文件保存的文件名

**下载类型：**单选，CSV 或者 TXT

**下载年月：**输入格式：yyyy-mm

**等待时间：**等待报表下载完成的时间





### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**下载文件路径：**下载文件的完整路径
