# 描述
该指令实现了自动切换站点，根据ASIN抓取买家之声页面的数据以及sku的详细数据的操作

# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称：**支持中文和英文简写，例：1.美国、2.US

**ASIN列表：**需要抓取买家之声的ASIN列表





### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**ASIN_data：**抓取的买家之声数据

**不存在的ASIN列表：**在页面上未找到的ASIN列表
