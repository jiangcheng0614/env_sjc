# 描述
该指令实现了亚马逊切换站点操作，自动点击切换过程出现的登录以及二次验证





![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1698024742392-93b97063-bf8f-4d70-84ad-b3c62e7e4a2c.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称：**支持中文和英文简写，例：1.美国、2.US

### 高级
无



## 指令输出
**无**



# 常见问题
1.二次验证验证码没有自动填充，请先进行配置.

如下情况需要到紫鸟浏览器给店铺配置自动二步验证

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1698024788174-32bd45b8-3434-4d58-8570-f693fdff552e.png)



2.暂不支持该站点

联系我添加站点映射表



3.当前店铺没有找到此站点

检查店铺站点信息是否填写正确
