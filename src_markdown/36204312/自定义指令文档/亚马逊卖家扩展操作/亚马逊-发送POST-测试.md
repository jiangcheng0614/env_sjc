# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，已打开亚马逊店铺的网页对象

**品牌帖子网址**：字符串，指定品牌帖子的网址，**页面路径**中**图三**页面的网址，可以直接访问到该品牌

示例：站点域名/brand-content-manager/profiles/品牌id

**文件路径**：字符串，图片路径

**标题：**字符串，帖子标题

**asin列表：**列表，asin列表

**开始时间**：字符串，格式：%Y-%m-%d %H:%M

**结束时间**：字符串，非必填，格式：%Y-%m-%d %H:%M



### 高级
无



## 指令输出
无
