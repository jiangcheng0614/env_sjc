# 描述
该指令实现了自动切换站点，抓取即将超出和已经超出预算的广告数据

# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称**：支持中文和英文简写，例：1.美国、2.US



### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**广告数据列表**：抓取的广告名称数据列表
