# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象





### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**字段列表**：抓取的数据对应的字段信息

**deal_data：**抓取的促销数据
