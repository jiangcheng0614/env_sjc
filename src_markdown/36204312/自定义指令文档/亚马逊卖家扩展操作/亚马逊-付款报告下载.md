# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称：**支持中文，例：1.美国

**账户类型：**单选、填写页面上显示的账户类型文本，如：标准订单，如需兼容多种语言，请参考下图查找对应的value来填写， 如：下图的 ` Visa 信用卡` 对应的 value 是 `CC_VISA`

![](https://cdn.nlark.com/yuque/0/2025/png/22242030/1738898155506-18c7b1c3-1632-4dd2-9cf8-cb5c4a0e9ac8.png)

**报告类型：**单选、填写：汇总 或者 交易

**保存文件夹：**下载文件保存的文件夹

**保存文件名：**下载文件保存的文件名





### 高级
**日期范围：**选填、不填默认上个月，填写格式：yyyy-mm-dd_yyyy-mm-dd

**报告标签：**选填、默认为空

**超时时间：**等待文件下载完成时间，单位秒



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**保存下载文件路径至：**下载的文件完整路径
