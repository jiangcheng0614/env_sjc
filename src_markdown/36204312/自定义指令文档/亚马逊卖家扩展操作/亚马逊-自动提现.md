# 参数配置
## 指令输入
### 常规
**网页对象**：已打开亚马逊店铺的网页对象

**站点名称：**支持中文和英文简写，例：1.美国、2.US

**最小提现金额：**大于等于此金额才执行提现操作





### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**提现金额：**本次成功提现的金额
