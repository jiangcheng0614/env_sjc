# 描述
该指令实现了自动切换站点，并在结算一栏页面抓取账户数据的操作

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1721719700436-ac3676c2-9579-4762-95ce-f8549c0e52ed.png)





# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，已打开亚马逊店铺的网页对象

**站点名称**：字符串，支持中文和英文简写，例：1.美国、2.US

**结算周期**：下拉框

+ 当前周期
+ 上一周期
+ 上两周期
+ 上三周期
+ 上四周期
+ xxx
+ 上九周期



### 高级
无



## 指令输出
**运行状态：**指令运行的结果，True 或 False

**异常信息：**如果指令运行失败，输出的异常信息

**账户数据：**抓取的账户数据
