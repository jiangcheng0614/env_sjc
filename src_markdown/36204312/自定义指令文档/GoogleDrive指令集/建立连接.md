# 注意事项
因为一些不可抗力的原因

首先要确保您所在的ip地址在<font style="color:#DF2A3F;">不开翻墙软件</font>的情况下，访问Google，不然是无法与GoogleDrive建立连接。

可以做个小小的测试，打开CMD，在里面输入ping google.com
