首先是查询语法

<font style="color:rgb(33, 44, 63);">Google Drive API的查询语法允许您构建更复杂的搜索查询，以便在Google Drive中搜索文件。以下是一些常用的查询语法示例：</font>

1. <font style="color:rgb(33, 44, 63);">按文件名称搜索：</font>
    - <font style="color:rgb(33, 44, 63);">name = 'file.txt'</font><font style="color:rgb(33, 44, 63);">：搜索文件名为"file.txt"的文件。</font>
    - <font style="color:rgb(33, 44, 63);">name contains 'keyword'</font><font style="color:rgb(33, 44, 63);">：搜索文件名中包含"keyword"的文件。</font>
    - <font style="color:rgb(33, 44, 63);">name = 'file.txt' and mimeType = 'text/plain'</font><font style="color:rgb(33, 44, 63);">：搜索文件名为"file.txt"且MIME类型为"text/plain"的文件。</font>
2. <font style="color:rgb(33, 44, 63);">按文件类型搜索：</font>
    - <font style="color:rgb(33, 44, 63);">mimeType = 'application/pdf'</font><font style="color:rgb(33, 44, 63);">：搜索MIME类型为"application/pdf"的PDF文件。</font>
    - <font style="color:rgb(33, 44, 63);">mimeType contains 'image/'</font><font style="color:rgb(33, 44, 63);">：搜索MIME类型以"image/"开头的图像文件。</font>
3. <font style="color:rgb(33, 44, 63);">按文件夹搜索：</font>
    - <font style="color:rgb(33, 44, 63);">'folderId' in parents</font><font style="color:rgb(33, 44, 63);">：搜索具有指定文件夹ID作为父文件夹的文件。</font>
    - <font style="color:rgb(33, 44, 63);">'root' in parents</font><font style="color:rgb(33, 44, 63);">：搜索位于根目录下的文件。</font>
4. <font style="color:rgb(33, 44, 63);">组合条件搜索：</font>
    - <font style="color:rgb(33, 44, 63);">name contains 'keyword' and mimeType = 'application/pdf'</font><font style="color:rgb(33, 44, 63);">：搜索文件名中包含"keyword"且MIME类型为"application/pdf"的PDF文件。</font>
5. <font style="color:rgb(33, 44, 63);">时间范围搜索：</font>
    - <font style="color:rgb(33, 44, 63);">modifiedTime > '2022-01-01T00:00:00'</font><font style="color:rgb(33, 44, 63);">：搜索修改时间晚于"2022-01-01 00:00:00"的文件。</font>
    - <font style="color:rgb(33, 44, 63);">createdTime < '2022-01-01T00:00:00'</font><font style="color:rgb(33, 44, 63);">：搜索创建时间早于"2022-01-01 00:00:00"的文件。</font>

<font style="color:rgb(33, 44, 63);">请注意，查询语法区分大小写，并且支持使用逻辑运算符（例如</font><font style="color:rgb(33, 44, 63);">and</font><font style="color:rgb(33, 44, 63);">、</font><font style="color:rgb(33, 44, 63);">or</font><font style="color:rgb(33, 44, 63);">、</font><font style="color:rgb(33, 44, 63);">not</font><font style="color:rgb(33, 44, 63);">）和括号来组合条件。</font>

<font style="color:rgb(33, 44, 63);">完整的查询语法文档可以在Google Drive API的官方文档中找到：</font>[https://developers.google.com/drive/api/v3/search-files](https://developers.google.com/drive/api/v3/search-files)
