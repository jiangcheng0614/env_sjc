# 一.功能描述
用于将列表数据生成一维的散点图表。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164551727-f0918280-11a3-4e9b-9d60-2edc94e8aab5.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164416267-3bd424d5-3a7f-4ca3-9b0d-53b92beaaf0d.png)

![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164400112-85e9792e-e333-4aa6-bf9e-2501fa171a5e.png)

## 1.输入参数
**X轴数据：**  
输入作为X轴的数据列表



**Y轴数据：**  
输入作为Y轴的数据列表



**图表保存路径：**  
输入图表需要保存的全路径或者文件夹路径



**预览：**

勾选：在电脑上展示图表，可以查看生成的图表样子

不勾选：不展示

### **高级**
**图表标题：**  
图表上方显示图表标题，默认为空。



**X轴标题：**  
图表设置X轴标题，默认为空。



**Y轴标题：**  
图表设置Y轴标题，默认为空。



**预览超时时间：**  
设置预览自动关闭时间，默认60s。

## 2.输出参数
**保存后路径：**  
输出图表保存到本地后的全路径。
