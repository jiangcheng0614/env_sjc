# 一.功能描述
用于将列表数据生成二维的散点图表。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164984872-e2fafb77-7c2b-44cb-99d2-a42acba98483.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164646381-2ca8166f-5107-41d8-89d9-18d15b8eebaa.png)

![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164653043-a6e16b94-4f04-440e-935b-411ca62ccfb4.png)

## 1.输入参数
**散1X轴数据：**  
输入作为第一维散点X轴的数据列表



**散1Y轴数据：**  
输入作为第一维散点Y轴的数据列表



**散2X轴数据：**  
输入作为第二维散点X轴的数据列表



**散2Y轴数据：**  
输入作为第二维散点Y轴的数据列表



**图表保存路径：**  
输入图表需要保存的全路径或者文件夹路径



**预览：**

勾选：在电脑上展示图表，可以查看生成的图表样子

不勾选：不展示

### **高级**
**图表标题：**  
图表上方显示图表标题，默认为空。



**X轴标题：**  
图表设置X轴标题，默认为空。



**Y轴标题：**  
图表设置Y轴标题，默认为空。



**预览超时时间：**  
设置预览自动关闭时间，默认60s。

## 2.输出参数
**保存后路径：**  
输出图表保存到本地后的全路径。
