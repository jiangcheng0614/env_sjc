# 一.功能描述
用于将列表数据生成三条线的线性图表。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680164325032-766fefed-40c9-4702-a56e-6191988597b7.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680163979059-e5193479-e8f8-4c22-87d5-7b8e9293e6d5.png)

![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680163986601-5cbeada5-47be-48ba-9683-c3c8b17ed523.png)

## 1.输入参数
**线1X轴数据：**  
输入作为第一条线X轴的数据列表



**线1Y轴数据：**  
输入作为第一条线Y轴的数据列表



**线2X轴数据：**  
输入作为第二条线X轴的数据列表



**线2Y轴数据：**  
输入作为第二条线Y轴的数据列表



**线3X轴数据：**  
输入作为第三条线X轴的数据列表



**线3Y轴数据：**  
输入作为第三条线Y轴的数据列表



**图表保存路径：**  
输入图表需要保存的全路径或者文件夹路径



**预览：**

勾选：在电脑上展示图表，可以查看生成的图表样子

不勾选：不展示

### **高级**
**图表标题：**  
图表上方显示图表标题，默认为空。



**X轴标题：**  
图表设置X轴标题，默认为空。



**Y轴标题：**  
图表设置Y轴标题，默认为空。



**线条1颜色：**  
设置第一条线的颜色，例如：red(红色),blue(蓝色),black(黑色)等。



**线条2颜色：**  
设置第二条线的颜色，例如：red(红色),blue(蓝色),black(黑色)等。



**线条3颜色：**  
设置第二条线的颜色，例如：red(红色),blue(蓝色),black(黑色)等。



**预览超时时间：**  
设置预览自动关闭时间，默认60s。

## 2.输出参数
**保存后路径：**  
输出图表保存到本地后的全路径。
