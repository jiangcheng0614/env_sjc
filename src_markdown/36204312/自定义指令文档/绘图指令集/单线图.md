# 一.功能描述
用于将列表数据生成单条线的线性图表。  
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680162127450-0ec6878a-8ee6-4881-aca3-dc58ec9bf138.png)

# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680161586847-68f89031-b8a8-4583-8d31-8fc0cbb61b3c.png)

![](https://cdn.nlark.com/yuque/0/2023/png/2214039/1680161734575-68f9d06a-003b-4624-b136-cf2970abb905.png)

## 1.输入参数
**X轴数据：**  
输入作为X轴的数据列表



**Y轴数据：**  
输入作为Y轴的数据列表



**图表保存路径：**  
输入图表需要保存的全路径或者文件夹路径



**预览：**

勾选：在电脑上展示图表，可以查看生成的图表样子

不勾选：不展示

### **高级**
**图表标题：**  
图表上方显示图表标题，默认为空。



**X轴标题：**  
图表设置X轴标题，默认为空。



**Y轴标题：**  
图表设置Y轴标题，默认为空。



**线条颜色：**  
设置线条的颜色，例如：red(红色),blue(蓝色)等。



**预览超时时间：**  
设置预览自动关闭时间，默认60s。

## 2.输出参数
**保存后路径：**  
输出图表保存到本地后的全路径。
