指令介绍：批量次数将数据插入数据库中，适合大量数据的插入。

## 描述
通过<font style="background-color:#D8DAD9;">数据库连接对象 </font>、<font style="background-color:#D8DAD9;">sql语句</font>和<font style="background-color:#D8DAD9;">插入数据</font>来对指定的数据库进行批量插入操作

## 参数配置
### 指令输入
#### 常规
**数据库连接对象: **_文本框 _; 与Hana数据库的连接对象

**sql语句: **_文本框 _; 批量插入的sql语句或者对应数据库表名

**插入数据: **_列表;插入的数据以元组组成的列表形式。_ 

#### 高级
**每批插入的数量: **以防止卡死，将采用分批的方式插入大数量的数据，默认为每批1000条

###  指令输出
**成功插入的个数:文本框;**大批量的数据一共插入了多少条
