## 描述
通过在SqlServer<font style="background-color:#D8DAD9;">服务器地址</font>，通过<font style="background-color:#D8DAD9;">用户名名称</font>和<font style="background-color:#D8DAD9;">用户名密码</font>来登录上数据库，再确认<font style="background-color:#D8DAD9;">数据库名称</font>来连接对应的数据库。返回数据库连接对象

## 参数配置
### 指令输入
#### 常规
**服务器地址: **_文本框 _; SQL server数据库所在的IP地址

**用户名名称: **_文本框 _; 登录SQL server数据库的用户名名称

**用户名密码: **_密码框 _; 登录SQL server数据库的用户名密码

**数据库名称: **_文本框 _; 对应账号下的某个数据库名称

#### 高级
**超时时间: **_字符串 _; SQL server数据库所在的IP地址

**端口号: **_字符串 _; SQL server所在的端口，默认为1433

**字符编码集: **_字符串 _; SQL server数据库默认使用的数据集

**TDS协议版本: **_字符串 _; _连接所使用的TDS协议版本，默认为None_

### 指令输出
**数据库连接对象:**SQL server数据库连接对象
