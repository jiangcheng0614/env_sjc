# 描述
保存当前网页的 Cookie 和 local_storage，用于下次登录

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1700646215032-fc1cf9f4-ae08-4296-9935-9db3fdd19103.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页自动化对象

**保存路径**：用于存储Cookie信息的文件路径





### 高级
无



## 指令输出
无



# 注意事项
1. 保存Cookie后，如果要切换账号登录，不要在页面上点击退出登录按钮，要用【Cookie退出登录】指令退出登录。否则保存的Cookie信息会失效，导致无法用该Cookie登录



2. Cookie一般都会有有效期，过了时间也会无法登录。需要重新保存Cookie，建议搭配自动化流程登录。每次切换账号先用Cookie登录，判断是否登录成功，如果不成功就用自动化流程登录，然后重新保存Cookie。
