# 描述
通过删除Cookie实现退出登录操作

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1700646638108-0d7805f6-a040-4abd-a09f-bf6d511ff0e9.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：网页自动化对象

**删除本地存储**：有的网站需要删除本地存储才能退出登录，有的网站不需要删除，一定要根据网站来决定，不要一股脑勾选



### 高级
无



## 指令输出
无

# 注意事项
1. 保存Cookie后，如果要切换账号登录，不要在页面上点击退出登录按钮，要用【Cookie退出登录】指令退出登录。否则保存的Cookie信息会失效，导致无法用该Cookie登录



2. Cookie一般都会有有效期，过了时间也会无法登录。需要重新保存Cookie，建议搭配自动化流程登录。每次切换账号先用Cookie登录，判断是否登录成功，如果不成功就用自动化流程登录，然后重新保存Cookie。
