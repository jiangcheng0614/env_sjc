# 描述
该指令实现获取**京搜客-数据报表-账户报表-账户**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677747742624-d14d1baf-b466-43b1-9c4d-94cfe0b726cf.png)

# 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677747657421-24d6ca0d-1987-486a-ad57-9ae3e386137b.png)



### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**开始时间：**时间字符串，格式为：'2022-01-02'

**结束时间：**时间字符串，格式为：'2022-01-02'  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下单或成交订单：**下拉选择框，选择下单订单或成交订单  
**分日报告：**勾选或不勾选

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
