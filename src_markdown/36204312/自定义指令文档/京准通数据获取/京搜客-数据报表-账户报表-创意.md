# 描述
该指令实现获取**京搜客-数据报表-账户报表-创意**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677746448049-93ea1278-0d85-45b1-8651-5e7456952c43.png)

### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**开始时间：**时间字符串，格式为：'2022-01-02'

**结束时间：**时间字符串，格式为：'2022-01-02'

**推广计划：**字符串，默认为'全部推广计划'

**推广创意：**字符串，默认为'全部推广创意'

**推广单元：**字符串，默认为'全部推广单元'  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下单或成交订单：**下拉选择框，选择下单订单或成交订单  
**分日报告：**勾选或不勾选

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
