# 描述
该指令实现获取**京速推-数据报表-账户报表-品类总览**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677729157089-221d5ed1-6dc3-4b8c-b396-926d948f1e0a.png)

# 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677728971708-2f53ee32-22b3-41f7-a9ad-76a5623f2c01.png)



### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**开始时间：**时间字符串，格式为：'2022-01-02'

**结束时间：**时间字符串，格式为：'2022-01-02'

**推广计划：**字符串，默认为’全部推广计划‘  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下部或成交订单：**下拉选择框，选择全部订单或成交订单  
**分日报告：**勾选或不勾选

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
