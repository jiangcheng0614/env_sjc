# 描述
该指令实现获取**京搜客-数据报表-账户报表-计划**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677758671337-9d3948b3-81d2-4537-94fe-84004f3d5539.png)

# 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677747257178-273cf3ac-943c-4d28-b9f4-3bbc7181c7bd.png)

### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**开始时间：**时间字符串，格式为：'2022-01-02'

**结束时间：**时间字符串，格式为：'2022-01-02'

**推广计划：**字符串，默认为'全部推广计划'  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下单或成交订单：**下拉选择框，选择下单订单或成交订单  
**分日报告：**勾选或不勾选

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
