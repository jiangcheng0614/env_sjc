# 描述
该指令实现获取数**据中心-行业分析-行业大盘-品牌流量排行**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677744749708-bc6b355c-3fab-4fc1-a6f5-f3b63712159d.png)

# 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677744172691-f3105d8e-81f3-46fc-87aa-da59c46129a3.png)





### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**日期：**下拉选择框，选择实时、昨日、近7天、近30天其中一个进行数据查询  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下单或成交订单：**下拉选择框，选择下单订单或成交订单  
**产品线：**下拉选择框，选择全选、京东展位、京东快车、京速推、购物触点、京东直投其中一个进行数据查询

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
