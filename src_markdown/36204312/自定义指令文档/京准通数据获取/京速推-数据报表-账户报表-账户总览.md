# 描述
该指令实现获取**京速推-数据报表-账户报表-账户总览**页面数据，并下载数据文件到指定位置  
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677812473549-13c3734e-f112-4c84-9ec3-aaf39d314b22.png)

# 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1677813401260-5fef5cb0-30f0-4eaa-9d5f-b4f11ca20ec4.png)



### 指令输入
**数据保存路径：**路径字符串，将获取到的数据保存至该文件夹下

**开始时间：**时间字符串，格式为：'2022-01-02'

**结束时间：**时间字符串，格式为：'2022-01-02'

**场景类型：**下拉选择框，默认为'全部场景类型'  
**转化周期：**下拉选择框，选择当天、1天、3天、7天、15天其中一个进行数据查询  
**点击或下单口径：**下拉选择框，选择点击或下单  
**含赠品或不含赠品：**下拉选择框，选择含赠品或不含赠品  
**下单或成交订单：**下拉选择框，选择下单订单或成交订单  
**分日报告：**勾选或不勾选

### 指令输出
**文件存储路径：**列表，存放下载的数据文件路径
