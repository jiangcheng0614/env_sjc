# 一、功能描述
实现抖音来客平台的自动登录

# ![](https://cdn.nlark.com/yuque/0/2024/png/39097414/1714276390809-7ef64f24-fa86-4e29-9c0c-11c43719de57.png)二、使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/39097414/1714276436071-1ea31c94-4858-48ab-8b9e-ca52a8934102.png)**登录方式是账号密码登录**

## 1.输入参数
**浏览器类型：**下拉框选择，支持谷歌、edge、火狐、IE、影刀、360安全浏览器，推荐使用谷歌浏览器

**账号：**用于登录的账号

**密码：**用于登录的密码

## 2.输出参数
**浏览器类型：**输出浏览器类型，例如Google Chrome浏览器、Microsoft Edge浏览器这种

**web_page：**输出网页对象，用于后续流程操作
