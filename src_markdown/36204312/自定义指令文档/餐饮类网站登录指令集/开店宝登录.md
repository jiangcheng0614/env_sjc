# 二.使用参数说![](https://cdn.nlark.com/yuque/0/2024/png/2214039/1714025172609-cb577905-ddf1-4f59-9ad5-f7f2178a6dc1.png)
## 1.输入参数
**浏览器**：下拉框选择，选择需要使用的浏览器类型

**用户名**：文本框，输入账号登录时的登录账号

**密码**：密码框，输入账号登录时的登录密码

**是否退出已登录**：勾选框，勾选时如果平台已经处于登录状态，先退出已经登录的账号，再进行账号登录，不勾选则不会退出已经登录号的账号。

## 2.输出参数
**web_page**：网页对象，输出登录完后的网页对象

**浏览器类型**：字符串，输出登录时使用的浏览器类型
