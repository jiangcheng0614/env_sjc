# 二、使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/40473584/1714098238359-f495b0b7-d153-4b0d-aa0c-fc7f81e240ef.png)

## 1.输入参数：
**浏览器类型：**下拉框选择，支持谷歌、edge、火狐、IE、360安全浏览器，默认是谷歌浏览器

**账号：**用于登录的账号

**密码：**用于登录的密码

## 2.输出参数
**browser_type：**输出浏览器类型，例如Google Chrome浏览器、Microsoft Edge浏览器这种

**web_page：**输出网页对象，用于后续流程操作

# 三、使用示例
