# 二、使用参数说明
![](https://cdn.nlark.com/yuque/0/2024/png/40473584/1714102982654-72a1b45c-c347-4027-96a2-07d220024477.png)

**登录方式是账号密码登录**

## 1.输入参数
**浏览器类型：**下拉框选择，支持谷歌、edge、火狐、IE、影刀、360安全浏览器，默认是谷歌浏览器

**账号：**用于登录的账号

**密码：**用于登录的密码

**邮箱账号：**用于接收手机验证码的邮箱账号，仅支持163、126、qq三种邮箱，填写形式如xxxxxx@163.com

**邮箱授权码：**邮箱授权码的获取方式自行百度

手机验证码转发需要在平台绑定的手机号上下载短信转发软件，转发到指定邮箱，再从指定邮箱里面获取邮件，从而进一步获取到验证码

ps:手机验证码转发到邮箱相关配置可参考（美团外卖商家中心登录）：[<font style="color:#3370FF;">手机短信验证码转发到邮箱</font>](https://ying-dao.feishu.cn/docx/TIjcdeXFPojltlxdikZcoSrin3d?from=from_copylink)

## 2.输出参数
**browser_type：**输出浏览器类型，例如Google Chrome浏览器、Microsoft Edge浏览器这种

**web_page：**输出网页对象，用于后续流程操作

# 三、使用示例
将指令拖到对应流程里面即可（账号、密码、邮箱账号、邮箱授权码都是必填的，运行前需要先填好）
