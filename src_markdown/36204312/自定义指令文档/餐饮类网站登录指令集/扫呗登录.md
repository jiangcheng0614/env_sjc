# 一.功能描述
实现扫呗平台自动登录。

# 二.使用参数说![](https://cdn.nlark.com/yuque/0/2024/png/2214039/1714025993186-225ad51c-93cc-4586-bde0-222742338c81.png)
## 1.输入参数
**浏览器**：下拉框选择，选择需要使用的浏览器类型

**用户名**：文本框，输入账号登录时的登录账号

**密码**：密码框，输入账号登录时的登录密码

**是否退出已登录**：勾选框，勾选时如果平台已经处于登录状态，先退出已经登录的账号，再进行账号登录，不勾选则不会退出已经登录号的账号。

## 2.输出参数
**web_page**：网页对象，输出登录完后的网页对象

**浏览器类型**：字符串，输出登录时使用的浏览器类型
