# 一.功能描述
获取列表中指定下标的元素值

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1679543667781-e4a9e09f-c68e-460d-9d88-d7593c8b4672.png)

# 二.使用参数说明
### 输入：
connection_pool：redis连接实例，是**启动redis连接**指令的返回值

key：需要查询的列表索引名称，格式为字符串  
index: 需要查询的列表指定下标位置，格式为整数

### 输出：
value:  返回列表中该查询下标的元素值，格式为字符串
