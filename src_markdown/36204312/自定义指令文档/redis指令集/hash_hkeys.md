# 一.功能描述
获取指定哈希表中的所有字段，并返回一个list

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1679573469840-67953d39-6bc6-44ce-9589-ba6672ff9524.png)

# 二.使用参数说明
### 输入：
connection_pool：redis连接实例，是**启动redis连接**指令的返回值

key：需要查询的hash的名称，格式为字符串

### 输出：
field_list:  返回该查询的hash所有的字段名列表，格式为列表
