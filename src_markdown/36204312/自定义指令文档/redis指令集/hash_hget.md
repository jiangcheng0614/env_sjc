# 一.功能描述
获取指定哈希表中的指定字段的值

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1679574101964-c3de9094-b278-4eef-8980-ed6237b99bb7.png)

# 二.使用参数说明
### 输入：
connection_pool：redis连接实例，是**启动redis连接**指令的返回值

key：需要查询的hash的名称，格式为字符串

field: 需要查询的hash的字段名称，格式为字符串

### 输出：
value:  返回该查询的hash的该字段名称的值，格式为字符串
