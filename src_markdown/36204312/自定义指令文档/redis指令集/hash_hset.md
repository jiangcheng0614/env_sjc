# 一.功能描述
将哈希表 key 中的字段 field 的值设为 value

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1679574411493-d8c9817d-2238-4b12-ad63-90591d4cb801.png)

# 二.使用参数说明
### 输入：
connection_pool：redis连接实例，是**启动redis连接**指令的返回值

key：需要设置的hash的名称，格式为字符串

field: 需要设置的hash的字段名称，格式为字符串

value:  返回设置的hash的该字段名称的值，格式为字符串
