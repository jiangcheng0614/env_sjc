# 一.功能描述
获取指定哈希表中的所有字段和值，并返回一个dict

![](https://cdn.nlark.com/yuque/0/2023/png/25504773/1679573740403-dd86d938-b6b6-460d-ab62-80cd0132f37e.png)

# 二.使用参数说明
### 输入：
connection_pool：redis连接实例，是**启动redis连接**指令的返回值

key：需要查询的hash的名称，格式为字符串

### 输出：
values_dict:  返回该查询的hash所有的字段和值，格式为字典
