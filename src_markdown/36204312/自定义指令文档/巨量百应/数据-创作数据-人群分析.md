# **输入**
### **web_page：**
任意一个已登录巨量百应达人后台的网页对象

### quick_date：
快捷日期；可填 近7天、近15天、近30天、近90天

### exact_date：
具体日期；可填202年14周（YYYY年W周）或2024年7月（YYYY年M月），此时快捷日期无效
