# **输入**
### **网页对象：**
任意一个已登录巨量百应达人后台的网页对象

### 快捷日期：
可填 近7天、近30天、自然周、自然月

### 准确日期：
可填YYYY年w周 （2024年14周）或  YYYY年M月（2024年7月）

### 另存后的文件名：
只要文件名前缀

### 保存路径：
文件保存路径

# 输出
### 完整路径列表：
一维列表，四个文件的完整路径
