# 描述
获取达人指标信息，成交金额、成交订单数、成交客单价、退款金额、预估佣金收入下的各自直播、短视频、图文、商品卡、其他 金额

# **输入**
### **网页对象：**
任意一个已登录巨量百应达人后台的网页对象

### 统计开始日期：
自定义中的统计开始日期；YYYY-MM-DD

### 统计结束日期：
自定义中的统计结束日期；YYYY-MM-DD

# 输出
### 概览数据：
二维列表，第一列是类别，第一行是列名

![](https://cdn.nlark.com/yuque/0/2024/png/33628300/1713161966737-0724ecdc-8c84-42e1-b117-886879d2fb5c.png)

### 达人口碑数据：
{'作者口碑分': 95, '内容口碑': 96, '商品口碑': 94, '服务口碑': 96}
