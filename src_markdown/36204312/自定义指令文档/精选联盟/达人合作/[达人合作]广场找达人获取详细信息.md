# 二.使用参数说明
![](https://cdn.nlark.com/yuque/0/2023/png/29236826/1681115788111-36b2a1ca-39e9-4b0a-b4b9-a46189026282.png)

## 1.输入参数
**网页对象：**  
选择需要操作的网页对象（请确保精选联盟or抖店处于登录状态）

**存储目录：**  
选填项。选择存放数据文件的文件夹，如不填写表示不输出Excel文件

**达人分类：**  
请填写您需要获取数据的达人分类（如不填写则不做该项选择操作）

**主推类目：**  
请填写您需要获取数据的达人主推类目（如不填写则不做该项选择操作）

输入完整的主推类目名称

**三十日销售额：**  
请填写您需要获取数据的达人30日销售额（如不填写则不做该项选择操作）

**场均销售额：**  
请填写您需要获取数据的达人场均销售额（如不填写则不做该项选择操作）

**单视频销售额：**  
请填写您需要获取数据的达人单视频销售额（如不填写则不做该项选择操作）

**粉丝量：**  
请填写您需要获取数据的达人粉丝量（如不填写则不做该项选择操作）

**有联系方**式**：**  
勾选表示只获取有联系方式的达人的详细数据

**概览数据范围：**  
目前支持：近7天，近30天，近3个月

### **高级**
**全局超时时间：**  
包含网页加载超时时间、等待元素超时时间等，请根据网速情况自行调整（部分指令超时时间不支持小数，所以请勿填写小数）

## 2.输出参数
**达人详细信息：**  
返回值为列表类型，因为是聚合数据，所以这里只提供相关数据列表

```json
[
  [表头行],
  [数据行],
  [数据行],
  ... ...
]
```

**文件保存路径：**  
如果填写了存储目录，可在此处接收文件保存路径
