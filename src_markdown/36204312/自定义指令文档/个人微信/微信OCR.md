# 描述
使用微信OCR对图片进行识别

![](https://cdn.nlark.com/yuque/0/2024/png/22242030/1707276443010-f925185b-a7ba-4fdb-9085-25493734a77d.png)



# 参数配置
## 指令输入
### 常规
**识别图片**：文件路径，待识别的本地图片路径

**输出格式：**下拉框

+ 详细：包含文字和坐标
+ 精简：只有文字

****

### 高级
无

****

## 指令输出
**识别结果：**OCR识别结果  




# 注意事项
1. 如果无法自动查找微信路径，需要使用 "[配置微信OCR](https://ydrpa.yuque.com/org-wiki-ydrpa-xtutvv/ga4dm6/config_wechat_ocr)" 指令配置相关路径
2. 如果不想安装整个微信，可以在安装微信的电脑上获取复制整个**微信OCR插件文件夹**以及**微信安装目录下的两个文件 mmmojo.dll 、mmmojo_64.dll **到使用OCR的电脑上，然后用  "[配置微信OCR](https://ydrpa.yuque.com/org-wiki-ydrpa-xtutvv/ga4dm6/config_wechat_ocr)" 配置一下路径即可
