# 参数配置
## 指令输入
### 常规
**微信窗口对象**：微信窗口对象

**返回数据格式：**

+ 列表类型的列表
+ 字典类型的列表

**截取消息数量：**会把抓取到的消息按指定数量截取，不填默认保留全部消息（需要注意的是，先执行高级输入参数的过滤，再执行截取操作）

### 高级

**保留自己消息：**保留自己消息

**保留对方消息：**保留对方消息
