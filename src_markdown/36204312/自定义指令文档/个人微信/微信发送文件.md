# 描述
微信给指定的联系人发送文件

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1679399630154-926aa95c-1270-4b0b-aa10-89f084056d53.png)

# 参数配置
## 指令输入
### 常规
**微信窗口对象**：微信窗口对象

**对方备注名：**对方备注名，如果没有备注则是微信昵称（非必填项，不填默认在当前聊天窗口发送消息）

**文件路径：**待发送的文件路径

****

### 高级
**发送方式**：根据自身需求选择发送的方式

+ 点击发送按钮
+ 快捷键：Ctrl+Enter
+ 快捷键：Enter
