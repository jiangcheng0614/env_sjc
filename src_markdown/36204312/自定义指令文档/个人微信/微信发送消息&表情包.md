# 描述
微信给指定的联系人发送文字消息和表情包

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1689570712336-3acf37a3-3398-4ef0-b12e-60116e941927.png)

# 参数配置
## 指令输入
### 常规
**微信窗口对象**：微信窗口对象

**对方备注名：**对方备注名，如果没有备注则是微信昵称（非必填项，不填默认在当前聊天窗口发送消息）

**消息：**待发送的消息和表情包，表情包要加上**<font style="color:#DF2A3F;">中括号</font>**或者加上**<font style="color:#DF2A3F;">%%</font>**，请查看下方的[使用案例](#GAMU6)

**@某人**：需要通过【获取@人员信息】 指令获取成员信息，然后将成员信息填与在这里。每个人的信息都是固定的，获取一次即可，自己保存永久使用

**@所有人：**如果你没有@所有人的权限，消息将会发送失败。选择了@所有人将不会执行@某人



### 高级
**发送方式**：根据自身需求选择发送的方式

+ 点击发送按钮
+ 快捷键：Ctrl+Enter
+ 快捷键：Enter



## 指令输出
**无**
