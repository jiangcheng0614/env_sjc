# 描述
先在微信上复制@的人员，然后运行此指令获取@人员信息，每个人的信息都是固定的，获取一次即可，自己保存永久使用

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1684922281258-c14d83d1-e65b-421f-95c4-7551f03296da.png)



# 参数配置
## 指令输入
### 常规
无



### 高级
无



## 指令输出
**保存人员信息列表至：**读取到的人员信息列表，列表的每一项是一个字典，字典的各个键值含义如下

+ name：人员名称
+ display：@当前人员时，显示的文字内容
+ wxid：当前人员的唯一id
