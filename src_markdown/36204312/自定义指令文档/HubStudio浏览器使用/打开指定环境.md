# 一、功能描述
通过接口的方式，打开HubStudio中某一个指定的环境

![](https://cdn.nlark.com/yuque/0/2023/png/561060/1680061900389-3e061d07-8eb0-4aa1-a16a-ebb8ddd3af15.png)

# 二、参数说明
**输入**

**环境ID：**HubStudio每个环境都有一个指定ID，可通过**获取环境列表**指令获取ID

**端口号：**HubStudio端口号默认是6873，也可以重新指定

**输出**

**运行结果：**访问请求的结果

# 四、常见问题
Q1：运行失败，没有打开环境

A1：打开环境需要一点等待时间。如果长时间打不开或者打开失败，检查端口号和环境ID
