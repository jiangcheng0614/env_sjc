# 一、功能描述
通过命令行的方式，启动HubStudio应用。只有执行此操作之后，才可以用其他的hubstudio指令。

![](https://cdn.nlark.com/yuque/0/2023/png/561060/1680061248004-8489731e-5e91-48bf-96be-fa455a4e9413.png)

# 二、参数说明
**参考链接：**[https://support.hubstudio.cn/7beb/935c/c681](https://support.hubstudio.cn/7beb/935c/c681)

**端口号**：HubStudio端口号默认是6873，也可以重新指定

**app_secret**：输入用户的app_secret

**app_id**:输入用户的app_id

**group_code**:输入用户的group_code

**安装目录**：输入HubStudio使用目录
