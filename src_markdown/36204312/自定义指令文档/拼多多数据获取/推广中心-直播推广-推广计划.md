# 描述
获取[推广中心-直播推广-推广计划](https://yingxiao.pinduoduo.com/marketing/main/center/broadcast/list)中的数据。

![](https://cdn.nlark.com/yuque/0/2023/png/12976921/1675327817656-30b3139f-fe33-4fd4-8813-064d3ad6d4ad.png)



# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/12976921/1675327822591-0f216eef-7462-49e7-b209-67eaf21c48b1.png)

## 输入
**网页对象:** 传入对应页面的网页对象<font style="color:#000000;">（当网页不为</font>指定页面<font style="color:#000000;">时，自动跳转到目标页面）。</font>

**<font style="color:#000000;">数据导出目录</font>**<font style="color:#000000;">：</font>保存当前数据的目录。

**查询起始时间: **推广数据的起始时间, 格式如2023-02-02, 为空则默认昨天。

**查询结束时间:** 推广数据的结束时间, 格式如2023-02-02, 为空则默认昨天。

## 输出
**<font style="color:#000000;">导出数据文件路径</font>**<font style="color:#000000;">：</font>输出文件路径<font style="color:#000000;">。</font>
