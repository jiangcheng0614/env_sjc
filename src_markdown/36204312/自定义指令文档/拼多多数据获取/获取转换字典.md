# 获取转换字典


![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1687743623256-0448a4bd-9f71-4b43-bb07-018c0d3e67f1.png)



### 描述


在打开的拼多多商家后台网页中获取转换字典，用于之后转换真实数据。详见[真实数据转换](https://ydrpa.yuque.com/org-wiki-ydrpa-xtutvv/ga4dm6/qft9lu0b3wb07kr5)。



### 配置项说明


#### 输入


**网页对象**：浏览器中打开的拼多多商家后台网页。



#### 输出


**转换字典**：作为转换真实数据的依据。



### 使用示例


**流程执行逻辑**：需要配合[真实数据转换](https://www.yuque.com/docs/share/e18dd97f-1f70-4599-83a4-d356d0476fcc)使用，详见[真实数据转换](https://www.yuque.com/docs/share/e18dd97f-1f70-4599-83a4-d356d0476fcc)。
