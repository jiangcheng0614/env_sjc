# 指令描述
该指令实现密码登录钉钉

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707040133046-0bd2d631-511d-41fc-a0ed-3c02e5643dad.png)

**输入参数 : **

**账号** : 钉钉登录账号

**密码** : 钉钉登录密码

**输出参数 : **

**钉钉密码登录布尔值**：返回钉钉密码登录状态
