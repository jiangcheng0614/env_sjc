# 指令描述
该指令实现钉钉发送文件给联系人

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707046051515-6e2950bd-925d-4512-9902-22b769ca1430.png)

**输入参数 : **

**接收人** : 消息接收人

**文件路径** : 待发送文件路径

**输出参数 : **

**钉钉发送文件布尔值**：返回钉钉发送文件状态
