# 指令描述
该指令实现钉钉发送信息给联系人

# 配置说明
**交互界面截图 : **

![](https://cdn.nlark.com/yuque/0/2024/png/35870027/1707041238025-36f1a829-f1ad-4d8f-90b2-487192249543.png)

**输入参数 : **

**消息内容** : 发送消息内容

**接收人 **: 钉钉消息接收人

**输出参数 : **

**钉钉发送消息布尔值**：返回钉钉发送信息状态
