# 一.功能描述
<font style="color:rgb(51, 51, 51);">输入值x，返回x的绝对值|x|。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676519695403-09e65af1-90de-40c3-87e8-210016aa66e8.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

输入需要取整的参数x，类型必须为数字number。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x_绝对值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回</font><font style="color:rgb(42, 43, 46);">x的绝对值，浮点类型。</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">-10的绝对值为10.0，返回值为float浮点型。</font>
