# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

必需，数字。如果 x 不是一个数字，返回 TypeError。如果值为 0 或负数，则返回 ValueError。

**base：**

可选，底数，此参数不填写则默认以<font style="color:rgb(51, 51, 51);">自然常数</font>e为<font style="color:rgb(51, 51, 51);">底数。</font>
