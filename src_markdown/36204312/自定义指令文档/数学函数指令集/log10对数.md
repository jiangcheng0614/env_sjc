# 一.功能描述
<font style="color:rgb(51, 51, 51);">对数底数为10，输入真数x，返回lgx。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676532775196-d44bdaf7-0569-4eaf-9576-d5ed9d57589e.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

必需，数字。如果 x 不是一个数字，返回 TypeError。如果值为 0 或负数，则返回 ValueError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**log10对数值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回lgx</font><font style="color:rgb(42, 43, 46);">值</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">真数x=100时，lg100=2 。</font>
