# 一.功能描述
<font style="color:rgb(42, 43, 46);">返回参数x的舍入值，例如n的值，它表示舍入到小数点的位数</font><font style="color:rgb(51, 51, 51);">。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676536105920-09c0f040-ab99-4d87-855e-b1e3c5a5f993.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

必需，数字。如果 x 不是一个数字，返回 TypeError。

**n：**

必需，int类型。保留的位数

### <font style="color:rgb(51, 51, 51);">输出</font>
**x浮点舍入值****<font style="color:rgb(51, 51, 51);">：</font>**

返回浮点数x的四舍五入值。

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">x为3.1415926时，n=2 四舍五入保留n位小数如下</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676536287873-d8d620dc-e092-4c36-8292-9faf96b551be.jpeg)

例：<font style="color:rgb(51, 51, 51);">n为-1，-2，-3，-4……时，对小数点左边第|n|位进行四舍五入 </font>

![](https://cdn.nlark.com/yuque/0/2023/png/1675452/1676536668158-3e4c6400-231a-480f-b628-b196951eef31.png)
