# 一.功能描述
<font style="color:rgb(51, 51, 51);">输入数值x，返回该数值的整数部分，即int(x)。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676535636822-c59a4add-5874-4d79-8130-f2c9333d58a3.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

必需，数字。如果 x 不是一个数字，返回 TypeError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x_整数值****<font style="color:rgb(51, 51, 51);">：</font>**

返回数值的整数部分，忽略小数部分。

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">输入x=1.23456，则返回值为1 。</font>
