# 一.功能描述
<font style="color:rgb(51, 51, 51);">输入两个数x、y，返回x除以y的余数。</font>

![](https://cdn.nlark.com/yuque/0/2023/png/1675452/1676526778892-f6d5e05d-048c-4162-9d98-85c90dbec122.png)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

被除数。必需，正数或负数。如果 x 不是一个数字，返回 TypeError。

**y：**

除数。必需，正数或负数。如果 y 不是一个数字，返回 TypeError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x除y余数值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回x除以y的余数</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">10除以3等于3余1，故10除以3取余为1。</font>
