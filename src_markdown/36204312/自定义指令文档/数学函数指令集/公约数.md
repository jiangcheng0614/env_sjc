# 一.功能描述
<font style="color:rgb(51, 51, 51);">返回x和y的最大公约数。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676527248009-00ffa293-4e5a-4fe1-96c2-1fd31ffd081a.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

必需，数字。如果 x 不是一个数字，返回 TypeError。

**y：**

必需，数字。如果 y 不是一个数字，返回 TypeError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x与y最大公约数****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回x和y的最大公约数。</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">8和6的公约数有1,2，故8和6的公约数输出最大公约数为2。</font>
