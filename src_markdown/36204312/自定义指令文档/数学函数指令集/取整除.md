# 一.功能描述
<font style="color:rgb(51, 51, 51);">x除以y，获取商的整数部分 。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676538898447-6b363757-6f23-452e-8373-729cc51ebfc3.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

x -- 必需，数字。如果 x 不是一个数字，返回 TypeError。

**y：**

y -- 必需，数字且不为0。如果 y 不是一个数字，返回 TypeError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x除y整数商值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回商的整数部分。</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">x=7, y=2   返回3 如下</font>
