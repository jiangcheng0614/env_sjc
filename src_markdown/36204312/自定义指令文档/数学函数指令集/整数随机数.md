# 一.功能描述
<font style="color:rgb(51, 51, 51);">随机返回x到y之间（包括x和y）的任意整数，其中x与y必须为整数（正负不限），且x<=y，否则报错。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676537741600-d711daad-1088-4336-a13c-2dd84e596e14.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

start -- 必需， 一个整数，指定开始值。

**y：**

stop -- 必需， 一个整数，指定结束值。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x与y之间随机整数值****<font style="color:rgb(51, 51, 51);">：</font>**

返回指定范围内的整数。

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">x、y分别输入1、10，则随机返回1,2,3,4,5,6,7,8,9,10中的任意一个整数</font>
