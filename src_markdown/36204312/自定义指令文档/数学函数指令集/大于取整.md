# 一.功能描述
<font style="color:rgb(51, 51, 51);">输入值x的小数部分大于0，则返回x的整数部分加一，即返回 int(x)+1 ；整数的大于取整还是该整数。</font>

![](https://cdn.nlark.com/yuque/0/2023/png/1675452/1676518567506-5cc82cc7-d471-407e-97cf-0a5370d351c2.png)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

输入需要取整的参数x，类型必须为数字number。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x_大于取整值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">输出</font><font style="color:rgb(42, 43, 46);">大于或等于x的最小整数值</font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">2的大于取整为2, 而2.54的大于取整为3，最终返回值为int整型。</font>
