# 一.功能描述
<font style="color:rgb(51, 51, 51);">输入一个自然数x，返回x的阶乘x! ；x可以是int整型或float浮点型，但必须为自然数，否则报错；返回值为int整型。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676526029684-20e60005-e88d-4802-8723-953fd89b82fb.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

输入参数x，必需，正整数。如果数字为负数或不是整数，则返回 ValueError。 如果值不是数字，则返回 TypeError。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x_阶乘值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">输出x的阶乘x! </font>

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">3的阶乘 3！=3×2×1=6 </font>
