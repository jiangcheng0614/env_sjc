# 一.功能描述
<font style="color:rgb(51, 51, 51);">随机返回项目sequence中的任意一个元素。sequence可以是列表，元组以及字符串等拥有下标的项目，不能是字典(dict)集合（set）等无序项目。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676538082235-ad76821a-595b-48f3-b746-ed6f0583b12a.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**序列：**

表示有序序列，可以是一个列表，元组或字符串。

### <font style="color:rgb(51, 51, 51);">输出</font>
**序列随机元素值****<font style="color:rgb(51, 51, 51);">：</font>**

返回序列随机元素

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">输入列表[1, 2, 3, "4", "5", "6"],则随机返回 1, 2, 3, "4", "5", "6" 中的任意一个：</font>
