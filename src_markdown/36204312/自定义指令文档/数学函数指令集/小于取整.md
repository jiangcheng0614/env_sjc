# 一.功能描述
<font style="color:rgb(51, 51, 51);">舍去输入值x的小数部分，返回x的整数部分，即对x取整返回int(x) 。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676526549579-caec1d63-ff83-4512-810f-8c6e1857f1e2.jpeg)

# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**x：**

输入需要取整的参数x，类型必须为数字number。

### <font style="color:rgb(51, 51, 51);">输出</font>
**x_小于取整值****<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">输出</font>小于或等于x的最大整数值，如果x是整数，则返回自身

<font style="color:rgb(42, 43, 46);">例：</font><font style="color:rgb(51, 51, 51);">3.1415926的小于取整为3 。</font>
