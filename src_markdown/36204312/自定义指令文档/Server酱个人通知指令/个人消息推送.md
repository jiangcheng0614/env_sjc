# **输入**
### **消息通道：**
微信服务号、安卓、ios

### Sendkey：
具体见下文

### 标题：
消息的标题，字符串

### 内容：
消息的具体内容，支持markdown格式



# 输出
### result：
是否发送成功

# 如何获取sendkey
1. [扫码（注册/登录）](https://sct.ftqq.com/login)
2. 复制sendkey，**不同的人有不同的Sendkey**![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1675907938569-06a03c71-e77a-4e8d-b7b0-b495b33c4c91.png#averageHue=%23faf7f7&clientId=uad40773a-9437-4&from=paste&height=303&id=u4551da1f&name=image.png&originHeight=303&originWidth=1165&originalType=binary&ratio=1&rotation=0&showTitle=false&size=30731&status=done&style=stroke&taskId=u0330ddaa-ade6-40a0-aa68-9d039227a19&title=&width=1165#averageHue=%23faf7f7&id=PTWLS&originHeight=303&originWidth=1165&originalType=binary&ratio=1&rotation=0&showTitle=false&status=done&style=none&title=)
3. 指令中的消息通道有微信服务号、安卓应用、iOS应用3种可选
4.  若选择**微信服务号**
    1. 在[https://sct.ftqq.com/forward](https://sct.ftqq.com/forward)中，点击「**方糖服务号**」，点击「**保存**」。 
5.  若选择**安卓** 
    1. 下载APK并安装，https://theseven.ftqq.com/download/sc9.apk（打开链接直接下载） 
    2. 输入或者扫码获得（官网右上角菜单中有「登入二维码」）SendKey 
    3. 点击登入按钮后在左上角菜单中点击「将当前设备注册为主设备」(如果没有看到，请点击「手动注册设备」) 注意：删除应用重新安装后要再次点击「将当前设备注册为主设备」；如果要切换回其他通道，可到[通道设置](https://sct.ftqq.com/forward)页面操作 
    4. 在[消息通道设置](https://sct.ftqq.com/forward)中点击「**官方Android版·β**」，点击「**保存**」。 
6.  若选择**ios** 
    1. [首先安装下载bark](https://apps.apple.com/cn/app/bark-%E7%BB%99%E4%BD%A0%E7%9A%84%E6%89%8B%E6%9C%BA%E5%8F%91%E6%8E%A8%E9%80%81/id1403753865) 
    2. 在[消息通道设置](https://sct.ftqq.com/forward)中点击「**Bark iOS**」， 
    3. 复制「**服务器连接**」（**注意不同Sevcer账号有不同的连接地址，不要用成同一个了，其实就是**[https://sctapi.ftqq.com?act=](https://sctapi.ftqq.com?act=)**sendkey**) 
    4. 在APP右上角的「**+**」在「**服务器地址**」处填入以上地址，并保存 
    5. 点击「**保存**」。
