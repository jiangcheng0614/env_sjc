# 描述
该指令实现了从本地上传文件到紫鸟网页端的操作

# 参数配置
## 指令输入
### 常规
**网页对象**：网页对象，紫鸟浏览器网页对象

**上传按钮**：网页元素，选择要点击上传按钮对应的网页元素

**上传文件路径**：字符串 | 列表， 待上传文件完整路径， 若要选择多个文件，切换到Python模式下，输入文件列

表，比如：[\"文件1路径\", \"文件2路径\"]

**保存文件名**：下载文件保存的文件名

**等待对话框超时时间：**整数，点击上传按钮后，等待上传对话框出现的超时时间，单位（秒）
