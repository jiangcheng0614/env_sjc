# 描述
该指令实现了自动打开紫鸟控制台，并登录**企业号**的操作

![](https://cdn.nlark.com/yuque/0/2025/png/22242030/1736909821992-86537030-3ac9-4a99-afaf-0e8bb009dba1.png)



# 参数配置
## 指令输入
### 常规
**企业名称**：企业名称

**企业用户名**：企业用户名

**密码**：密码

**是否填写账号密码**：如果账号已经勾选了记住密码，可以不用填写，直接点击登录

**是否退出登录：**是否重新登录

### 高级
无

# 注意事项
1. 当前指令支持V5 V6 两个大版本
2. V5如果插件配置了正确的启动路径，在使用登录的时候会自动打开工作台，V6暂时不支持自动打开
