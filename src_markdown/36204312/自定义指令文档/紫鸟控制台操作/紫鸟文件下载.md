# 描述
该指令实现了紫鸟浏览器网页端的文件下载操作

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1695023438331-e3a68d3d-c125-414c-99be-a7c4b031c33e.png)



# 参数配置
## 指令输入
### 常规
**网页对象**：紫鸟浏览器网页对象

**下载按钮**：触发下载文件的元素按钮

**保存文件夹**：下载文件保存的文件夹

**保存文件名**：下载文件保存的文件名

**超时时间：**等待文件下载的时间，超出这个时间将会抛出异常

****

### 高级
默认下载文件保存位置：紫乌控制台设置的默认下载路径，不填会自动从配置文件获取，如果配置文件获取不到可以手动填写

是否询问每个文件的保存位置：紫乌控制台设置的是否询问每个下载文件，不填会自动从配置文件获取，如果配晋文件获取不到可以手动填写



:::info
对于高级参数的解释：

由于紫鸟下载文件到非环境指定位置，会有一个提示弹窗导致流程卡住。当前指令的处理方式是：先下载到环境指定位置，再移动到目标位置。所以就需要先获取紫鸟指定的默认下载位置

有的环境可以直接从配置文件获取紫鸟指定的默认下载位置，有的环境不可以。如果获取不到则会在工作台的设置页面获取，如果不想在页面上获取，则就需要手动在高级页面传入紫鸟指定的默认下载位置

:::

## 指令输出
**download_file_name**：下载的文件的完整路径
