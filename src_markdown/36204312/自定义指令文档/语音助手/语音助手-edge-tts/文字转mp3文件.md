# 参数配置
## 指令输入
### 常规
**角色：**说话的角色，F结尾的是女声，M结尾的是男声

**文字：**需要播放声音的文字

**音频文件路径：**保存的音频文件路径



### 高级
**语速：**请输入：-50~50，默认为0

**音量：**请输入：-50~50，默认为0，0并不代表没有声音



## 指令输出
无
