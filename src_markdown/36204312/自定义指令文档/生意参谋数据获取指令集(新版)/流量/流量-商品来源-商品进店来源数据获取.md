# 描述
该指令实现了【生意参谋】->【流量】->【商品来源】->【商品进店来源数据获取】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23081768/1676895483083-d1b43756-7abf-4a50-b6b6-639950349d9f.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期

保存文件夹：请输入数据文件保存的文件夹路径

商品ID列表：请输入需要查询的商品ID列表

## 输出
输出文件路径：输出数据文件的路径
