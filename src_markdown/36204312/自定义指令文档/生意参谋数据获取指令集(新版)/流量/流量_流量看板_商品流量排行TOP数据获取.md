# 描述
该指令实现了【生意参谋】->【流量】->【流量看板】->【商品流量排行TOP数据获取】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23081768/1676887037983-3ef40a93-e650-4b03-aa2f-7e71d3426230.png)

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677131525095-27e123ba-8185-47b7-8c13-0ba802d53561.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期

保存文件夹：请输入数据文件保存的文件夹路径

## 输出
商品流量排行TOP10_数据列表：输出的商品流量排行TOP10的数据

输出文件路径：输出数据文件的路径
