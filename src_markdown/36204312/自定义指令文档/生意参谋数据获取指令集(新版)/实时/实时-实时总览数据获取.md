# 描述
该指令实现了【生意参谋】->【实时】->【实时概况】->【实时总览】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677121635964-db1cfd89-607a-4e49-9d9f-f8eb4c864397.png)

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677121675377-53c22f40-674c-4884-a296-64762478438d.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

终端选择 : 数据终端类型的选择 

所有终端 / 无线端 / PC端

## 输出
输出文件路径：输出数据文件的路径

执行状态 : 伴随输入 复选框 开启页面统计时间校验 输出成功或者异常内容 供开发者捕获判断 .
