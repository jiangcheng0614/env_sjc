# 描述
该指令实现了【生意参谋】->【交易】->【交易概况】->【品牌构成】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677135800019-9cbc986c-c0ab-4e2b-b879-e593e8872e16.png)

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677135859863-7057e4f6-16ff-403d-8593-a2cbb46a0151.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

终端选择 : 全部 / PC / 无线 

保存文件夹：请输入数据文件保存的文件夹路径

## 输出
输出文件路径 : 下载文件路径信息
