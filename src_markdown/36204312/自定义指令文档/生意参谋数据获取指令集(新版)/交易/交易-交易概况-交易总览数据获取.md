# 描述
该指令实现了【生意参谋】->【交易】->【交易概况】->【终端构成】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

高级：

快捷日期：下拉框(日，周，月)

日：格式YYYY-MM-DD，即选择当天

周：格式YYYY-MM-DD，选择包含日期的那一周

月：格式YYYY-M(M)，选择指定月。2024年一月即2024-1. 2024年12月即2024-12

## 输出
输出文件路径 : 下载文件路径信息
