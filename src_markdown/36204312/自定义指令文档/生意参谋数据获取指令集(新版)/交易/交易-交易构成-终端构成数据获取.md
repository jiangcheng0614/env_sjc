# 描述
该指令实现了【生意参谋】->【交易】->【交易构成】->【终端构成】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677132058566-6f1d53e9-a90e-4b22-949c-4494b8963a0c.png)

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677132222140-bf3d6ddf-75df-4e38-a984-3f16e696b7f6.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

## 输出
输出文件路径 : 下载文件路径信息
