# 描述
该指令实现了【生意参谋】->【服务】->【客服绩效】->【专项分析】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677065354369-2dc4fdcf-a734-40d8-af74-e2143ded9f18.png)

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677065364976-1ba57c7e-81f1-4e89-894b-9d04b3ce3e8f.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

分析类型 : 

客单价分析/咨询接待分析/接待评价分析/商品销售分析/预售分析/成功退款分析/分时接待分析/值班分析

<font style="color:#DF2A3F;">开启页面统计时间校验: 勾选后会在日期选择之后 判断页面显示的统计时间 , 如果跟用户输入查询日期不一致, 会结束运行, 返回一个执行状态 可供用户调用后进行捕获判断 . </font>

## 输出
输出文件路径：输出数据文件的路径

执行状态 : 伴随输入 复选框 开启页面统计时间校验 输出成功或者异常内容 供开发者捕获判断 .
