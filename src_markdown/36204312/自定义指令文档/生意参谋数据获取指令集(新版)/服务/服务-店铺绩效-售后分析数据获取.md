# 描述
该指令实现了【生意参谋】->【服务】->【店铺绩效】->【售后分析】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677064475938-6dbc0b84-af12-4944-871e-846796631beb.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

售后退款处理类型 : 售后分析退款类型选择  店铺/未发货仅退款/已发货仅退款/收到货仅退款/仅退款/退费退款 这六种类型![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677064520590-89e22979-737d-4cb6-b41b-60e534e9df27.png)

<font style="color:#DF2A3F;">开启页面统计时间校验: 勾选后会在日期选择之后 判断页面显示的统计时间 , 如果跟用户输入查询日期不一致, 会结束运行, 返回一个执行状态 可供用户调用后进行捕获判断 . </font>

## 输出
输出文件路径：输出数据文件的路径

执行状态 : 伴随输入 复选框 开启页面统计时间校验 输出成功或者异常内容 供开发者捕获判断 .
