# 描述
该指令实现了【生意参谋】->【服务】->【绩效明细】->【流失明细】部分的数据获取，并将数据写入到Excel文件保存到指定路径下；

![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677068794904-651a6342-2010-4c1b-bc54-a799ed60c9da.png)

# 配置说明
![](https://cdn.nlark.com/yuque/0/2023/png/23170566/1677068799490-5802e25f-8cce-4b61-bf3e-2aca27a665fc.png)

## 适用的浏览器类型
1. Google Chrome浏览器

## 输入
查询日期：请输入需要查询的日期 支持格式 : 当前页面的快捷日期(日,周,月,近7天 .. ) 和自定义单日日期 2023-01-01

保存文件夹：请输入数据文件保存的文件夹路径

质量明细类型 : 

接待评价明细 / 慢响应明细 / 长接待明细 

<font style="color:#DF2A3F;">开启页面统计时间校验: 勾选后会在日期选择之后 判断页面显示的统计时间 , 如果跟用户输入查询日期不一致, 会结束运行, 返回一个执行状态 可供用户调用后进行捕获判断 . </font>

## 输出
输出文件路径：输出数据文件的路径

执行状态 : 伴随输入 复选框 开启页面统计时间校验 输出成功或者异常内容 供开发者捕获判断 .
