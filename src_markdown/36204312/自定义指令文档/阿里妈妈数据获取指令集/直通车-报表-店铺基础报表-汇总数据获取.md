##### 概述
当前版本指令实现了阿里妈妈-直通车-报表-店铺基础报表下各个子报表的汇总数据的获取，可实现自定义转化周期选择，数据日期的选择。

**注：当前版本指令仅支持Chrome浏览器**，使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡。

##### 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1680773843739-bfde76af-e0b0-4c90-b19b-dde0293aacc6.png)

+ **web_page**
    - 类型：网页对象
    - 需要进行数据获取页面的网页对象
+ **报表类型**
    - 类型：字符串
    - 当前版本指令支持基础报表下，所有子报表的汇总数据获取，例如：账户报表，计划报表，单元报表...
+ **转化周期**
    - 类型：字符串
    - 可选参数： 1天转化数据，3天转化数据，7天转化数据，15天转化数据，15天累计数据，30天累计数据
    - 注：**注意参数格式需要和可选参数格式完全一致**，否则会导致周期选择失败
+ **日期**
    - 类型： 字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
    - 当前版本指令支持以下快捷日期列表中的任一快捷日期的选择
    - 快捷日期列表 = ['昨天','前天','过去7天','上周','过去14天','过去30天','上月','过去90天','过去半年']
    - **注意： 快捷日期若有空格请勿省略，否则会导致快捷日期选择失败**

##### 指令输出
result

+ 类型：列表
+ 例：[[字段名],[数据]]
+ **注**：需要获取的数据字段需在指令运行前进行手动配置
