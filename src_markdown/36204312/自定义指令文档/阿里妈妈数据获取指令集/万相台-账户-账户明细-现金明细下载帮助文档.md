##### 概述
当前版本指令实现了万象台-账户-账户明细-现金明细的自定义下载 ，可实现对收支类型，交易类型，日期类型，日期范围及文件保存路径的自定义。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 参数配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1680233935375-e100ef37-3b1a-411c-befd-e123cb8a1d40.png)

+ wab_page
    - 类型：网页对象
    - 描述：万象台-账户-账户明细-现金明细所在页面网页对象
+ 收支类型
    - 类型：字符串
    - 可选参数
        * 全部收支类型
        * 收入
        * 支出
+ 交易类型
    - 类型：字符串
    - 可选参数
        * 全部交易类型
        * 充值
        * 退款
        * 扣款
        * 转账
        * 赔付
        * 冻结
        * 解冻
+ 日期类型
    - 类型：字符串
    - 可选参数
        * 记账时间
        * 交易日期
+ 日期范围
    - 类型：字符串
    - 格式：yyyy-mm-dd~yyyy-mm-dd，例：2023-03-15~2023-03-20
+ 文件保存路径
    - 类型：字符串
    - 描述：文件保存至本地的路径

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件保存至本地的文件路径
+ 例：C:\Users\wangshu\Desktop\万象台-报表-账户明细-现金明细-全部收支类型-全部交易类型-记账时间-2023-03-12~2023-03-22.xls
