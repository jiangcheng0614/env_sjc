##### 概述
当前版本指令实现了对引力魔方_报表_地域报表的，营销场景，报表维度，时间维度，时间范围，转化周期，文件存储路径等参数的自定义及报表下载。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677207122407-fa468418-8469-4f16-ac1f-51541b0dc4dc.png)

+ web_page
    - 类型：网页对象
    - 需要进行数据获取页面的网页对象
+ 效果
    - 类型：字符串
    - 可选参数
        * 点击效果
        * 展现效果
+ 转化周期
    - 类型：字符串
    - 可选参数
        * 转化效果周期1天
        * 转化效果周期3天
        * 转化效果周期7天
        * 转化效果周期15天
+ 日期范围
    - 类型：字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
+ 文件保存路径
    - 类型：字符串
    - 文件下载至本地存储路径
    - **注：**若为空，则默认存储位置为桌面

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件下载至本地的文件路径
+ 例：C:\Users\wangshu\Desktop\引力魔方_报表_地域报表_汇总数据_2023-02-01~2023-02-13.xlsx
