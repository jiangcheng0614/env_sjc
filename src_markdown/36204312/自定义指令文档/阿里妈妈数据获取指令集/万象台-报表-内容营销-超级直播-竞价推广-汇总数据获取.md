##### 概述
当前版本指令实现了对万象台_报表_内容营销_超级直播_竞价推广_汇总数据获取的转化周期，日期范围，数据类型等参数的自定义及数据下载。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677207091567-83f5692e-b237-4006-b7d3-88cda10756ae.png)

+ web_page
    - 类型：网页对象
    - 需要进行数据获取页面的网页对象
+ 转化周期
    - 类型：字符串
    - 可选参数
        * 转化效果周期 - 3天
        * 转化效果周期 - 7天
        * 转化效果周期 - 15天
        * 转化效果周期 - 30天
+ 日期范围
    - 类型：字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件下载至本地的文件路径
