##### 概述
当前版本指令实现了品销宝-账户-账户明细-财务记录的自定义下载 ，可实现对报表类型，资产类型，流水类型，日期范围及文件保存路径的自定义。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 参数配置
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1680245100255-8c96b1e7-a1d8-4451-bfa9-861e81f2076a.png)

+ wab_page
    - 类型：网页对象
    - 描述：品销宝-账户-账户明细-现金明细所在页面网页对象
+ 报表类型
    - 类型：字符串
    - 可选参数
        * 明星店铺
        * 明星PUSH
        * 品牌首推
+ 资产类型
    - 类型：字符串
    - 可选参数
        * 全部资金类型
        * 现金账户
        * 赠款账户
        * 保证金/冻结账户
+ 流水类型
    - 类型：字符串
    - 可选参数
        * 全部流水类型
        * 收入
        * 支出
+ 日期范围
    - 类型：字符串
    - 格式：yyyy-mm-dd~yyyy-mm-dd，例：2023-03-15~2023-03-20
+ 文件保存路径
    - 类型：字符串
    - 描述：文件保存至本地的路径，若为空，则默认将文件下载到桌面文件夹

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件保存至本地的文件路径
+ 例：C:\Users\wangshu\Desktop\品销宝-报表-账户明细-现金明细-明星店铺-全部资金类型-全部流水类型-2023-03-12~2023-03-22.xls
