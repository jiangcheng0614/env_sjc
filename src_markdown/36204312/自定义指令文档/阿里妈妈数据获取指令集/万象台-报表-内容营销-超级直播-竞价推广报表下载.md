##### 概述
当前版本指令实现了对万象台_报表_内容营销_超级直播_竞价推广_报表的转化周期，日期范围，主体类型，报表名称等参数的自定义及数据下载。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677207086207-653e79c1-2267-4989-a595-424945a2aa6b.png)

+ web_page
    - 类型：网页对象
    - 需要进行数据获取页面的网页对象
+ 转化周期
    - 类型：字符串
    - 可选参数
        * 转化效果周期 - 3天
        * 转化效果周期 - 7天
        * 转化效果周期 - 15天
        * 转化效果周期 - 30天
+ 日期范围
    - 类型：字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
+ 报表名称
    - 类型：字符串
    - 可选参数
        * 推广
        * 定向
        * 创意
+ 主体类型
    - 类型：字符串
    - 可选参数
        * 全部主体类型
        * 直播间
        * 直播看点
+ 文件保存路径
    - 类型：字符串
    - 描述：文件下载到本地的存储路径
    - **注：**参数可为空，若文件保存路径为空，则默认将文件下载到用户桌面

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件下载至本地的文件路径
+ 例：C:\Users\wangshu\Desktop\万象台_报表_超级直播_推广_2023-02-01~2023-02-13.xls
