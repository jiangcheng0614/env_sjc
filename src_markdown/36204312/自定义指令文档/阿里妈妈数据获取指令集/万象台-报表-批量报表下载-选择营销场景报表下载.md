##### 概述
当前版本指令实现了对万象台_报表_批量报表下载_选择营销场景的数据层级，营销场景，报表维度，时间维度，时间范围，转化周期，文件存储路径等参数的自定义及报表下载。

**注：当前版本指令仅支持Chrome浏览器**，**使用本指令获取数据时，请确保当前页面为指定报表页面，且无其他元素遮挡**。**chrome浏览器需在【设置】→【下载内容】→【****<font style="color:rgb(32, 33, 36);">下载前询问每个文件的保存位置</font>****设置】选择开启，否则会导致文件下载异常。**

##### 配置项说明
![](https://cdn.nlark.com/yuque/0/2023/png/27270471/1677207106230-503430f2-9cf2-4046-b919-dbc87456acab.png)

+ web_page
    - 类型：网页对象
    - 需要进行数据获取页面的网页对象
+ 营销场景
    - 类型：字符串
    - 可选参数
        * 全选
        * 拉新快
        * 会员快
        * 获客易
        * 测款快
        * 上新快
        * 货品加速
        * 活动加速
+ 转化周期
    - 类型：字符串
    - 可选参数
        * 7天
        * 15天
        * 30天
+ 日期范围
    - 类型：字符串
    - 自定义日期格式：yyyy-mm-dd~yyyy-mm-dd，例：2022-11-12~2022-12-12
+ 数据层级
    - 类型：字符串
    - 可选参数
        * 汇总数据
        * 分计划数据
+ 报表维度
    - 类型：字符串
    - 可选参数
        * 不区分
        * 主体
        * 创意
        * 优化目标
+ 时间维度
    - 类型：字符串
    - 可选参数
        * 汇总数据
        * 分日数据
+ 文件保存路径
    - 类型：字符串
    - 文件下载至本地存储路径
    - **注：**若为空，则默认存储位置为桌面

##### 指令输出
file_path

+ 类型：字符串
+ 描述：文件下载至本地的文件路径
+ 例：C:\Users\wangshu\Desktop\万象台_报表_批量报表下载_选择营销场景_全选_2023-02-01~2023-02-13.xls
