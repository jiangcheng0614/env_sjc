# 一.功能描述
<font style="color:rgb(51, 51, 51);">对未加密的目标PDF文件进行加密，并复制生成一个新的PDF文件。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676877075663-9964757f-6a33-4848-81f3-72848f3a97e2.jpeg)



# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**<font style="color:rgb(51, 51, 51);">PDF文件路径：</font>**

传入待加密的目标PDF文件的路径，输入为字符串，也可点击输入框右侧的选择文件进行选择。

**<font style="color:rgb(51, 51, 51);">设置打开密码：</font>**

<font style="color:rgb(51, 51, 51);">设置生成的PDF文件的打开密码，</font>注：密码为数字或字母。 

**PDF保存路径：**

加密后pdf保存的路径，若为空, 则在安装目录下shadowbot中的temp文件夹。

**PDF保存名称：**

加密后pdf的保存名称，输入为字符串，若不填写,则随机生成8位名称

### <font style="color:rgb(51, 51, 51);">输出</font>
返回加密PDF路径**<font style="color:rgb(51, 51, 51);">：</font>**

<font style="color:rgb(51, 51, 51);">返回复制生成的加密PDF文件的全路径。</font>

例如 'E:/test/demo.pdf'
