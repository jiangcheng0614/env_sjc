# 描述
适用旺店通<font style="color:#DF2A3F;">企业版</font>接口查询订单数据，由于相关规定接口查不到淘系订单数据。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1687159476229-8e32bc0d-a437-4050-8504-c05bd1e0845a.png)

### 业务参数：
要求字典格式；

以修改时间范围查询为示例，查询最后**创建日期**在2023-03-26 16:39:00到2023-03-26 17:39:00 的**待审核**退换单。

<font style="color:rgb(51, 51, 51);">time_type：时间类型</font>**<font style="color:rgb(51, 51, 51);">，</font>**<font style="color:rgb(51, 51, 51);">start_time</font>：时间开始，<font style="color:rgb(51, 51, 51);">start_time</font>：时间结束，<font style="color:rgb(51, 51, 51);">process_status</font>：退换单处理状态

**{"****<font style="color:rgb(51, 51, 51);">time_type</font>****":"2", "****<font style="color:rgb(51, 51, 51);">start_time</font>****":"2023-03-26 16:39:00", "****<font style="color:rgb(51, 51, 51);">end_time</font>****":"2023-03-29 17:39:00", "****<font style="color:rgb(51, 51, 51);">process_status</font>****":"20"}**

更多业务参数请查看开放平台官网[链接](https://dz.wangdian.cn/qyb/open/apidoc)
