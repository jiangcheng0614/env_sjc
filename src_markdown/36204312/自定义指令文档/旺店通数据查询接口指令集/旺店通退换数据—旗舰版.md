# 描述
适用旺店通<font style="color:#DF2A3F;">旗舰版</font>接口查询订单数据，由于相关规定接口查不到淘系订单数据。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1685591784129-218e60ad-b2e7-47a5-a617-d1f3ad4d0201.png)



# **输入**
### appkey：
详见下文

### appsecret：
详见下文

### 卖家账号：
详见下文

### 环境：
可选**正式环境**和**测试环境**，旺店通开放平台注册后会提供测试环境，请先在测试环境调试。



### 业务参数：
要求字典格式；

以修改时间范围查询为示例，查询最后**修改日期**在2023-03-26 16:39:00到2023-03-26 17:39:00 的**未审核**退换单。

modified_from：修改时间开始，modified_to：修改结束结束，status：退换状态

{"modified_from":"2023-03-26 16:39:00",  "modified_to":"2023-03-26 17:39:00",  "status":20}

更多业务参数请查看开放平台官网[链接](https://dz.wangdian.cn/qjb/open/apidoc/doc?path=aftersales.refund.Refund.search)

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1686900994374-fc834af5-911b-4d8c-a481-6f18ac4d1d41.png)
