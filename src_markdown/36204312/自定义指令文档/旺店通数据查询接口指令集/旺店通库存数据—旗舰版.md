# 描述
适用旺店通<font style="color:#DF2A3F;">旗舰版</font>接口查询数据，增量查询库存数据，主要按时间查询

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1685585086694-f7385d98-fead-4d5a-a103-1280c41de64d.png)



# **输入**
### appkey：
详见下文

### appsecret：
详见下文

### 卖家账号：
详见下文

### 环境：
可选**正式环境**和**测试环境**，旺店通开放平台注册后会提供测试环境，请先在测试环境调试。



### 业务参数_查询参数：
要求字典格式；

以时间范围查询为示例，查询2023-03-26 16:39:00到2023-03-27 16:39:00 商品库存。

**注意**因为是增量查询，如果时间范围内某个商品库存无变动，则返回结果中不包含该商品。

根据时间范围查询时，必须要**start_time **和 **end_time 两个键值对，**时间范围最大跨度为30天

{"start_time":"2023-03-26 16:39:00","end_time":"2023-03-27 16:39:00"}

更多业务参数请查看开放平台官网[链接](https://dz.wangdian.cn/qjb/open/apidoc/doc?path=wms.StockSpec.search)
