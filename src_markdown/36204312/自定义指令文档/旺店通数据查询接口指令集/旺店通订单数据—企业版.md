# 描述
适用旺店通<font style="color:#DF2A3F;">企业版</font>接口查询订单数据，由于相关规定接口查不到淘系订单数据。

![](https://cdn.nlark.com/yuque/0/2023/png/33628300/1682599660374-5c8ede78-beac-4489-87a1-7b3906b5c817.png)



# **输入**
### appkey：
详见下文

### appsecret：
详见下文

### 卖家账号：
详见下文

### 环境：
可选**正式环境**和**测试环境**，旺店通开放平台注册后会提供测试环境，请先在测试环境调试。



### 业务参数：
要求字典格式；

以时间范围查询为示例，查询2023-03-26 16:39:00到2023-03-26 17:39:00 订单。

根据时间范围查询时，必须要**start_time **和 **end_time 两个键值对，**时间范围最大跨度为**1小时**

{"start_time":"2023-03-26 16:39:00","end_time":"2023-03-26 17:39:00"}

更多业务参数请查看开放平台官网[链接](https://dz.wangdian.cn/qyb/open/apidoc)
