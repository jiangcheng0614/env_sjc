# 描述
该指令可以自动选择小红书千帆平台数据栏目下的时间

![](https://cdn.nlark.com/yuque/0/2024/png/5368516/1715831753619-b465b728-28d4-4a5f-afe0-f770501fe0d5.png)

# 指令说明
![](https://cdn.nlark.com/yuque/0/2024/png/5368516/1715917719261-90ed4d8a-b7a9-467d-a65d-ba15efadc807.png)

网页对象

日期所在的元素

开始时间：根据时间可以填写**近1日、近7日、近30日，YYYY-MM-DD**格式的日期

结束时间



比如，您想要点选**<font style="color:rgb(38, 38, 38);">经营数据概览</font>**<font style="color:rgb(38, 38, 38);">的时间选择框，那边</font>**<font style="color:rgb(38, 38, 38);">日期所在的元素</font>**<font style="color:rgb(38, 38, 38);">则传入该行的元素（图中已标准红框）</font>

![](https://cdn.nlark.com/yuque/0/2024/png/5368516/1715918087312-2d71f35b-f79b-4e04-abae-d672f988b616.png)

指令会根据填写的内容自动判断时间选择
