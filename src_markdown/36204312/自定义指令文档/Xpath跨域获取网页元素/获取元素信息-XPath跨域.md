# 参数配置
## 指令输入
### 常规
**iframe对象：**通过【初始化iframe】指令或者【切换iframe】指令获取的输出对象，也可以传入网页对象

**xpath：**目标元素的xpath，支持传入数组，可以实现精确切入多层iframe查找元素。传入数组，则不会全局查找

**操作：**目标元素的各类信息

**基于当前iframe全局查找：**

**勾选：**会基于当前iframe对象以及所有子iframe对象查找元素

**不勾选：**只在当前iframe对象查找元素



### 高级
**超时时间:**元素查找超时时间



## 指令输出
**attribute:**目标元素的元素信息
