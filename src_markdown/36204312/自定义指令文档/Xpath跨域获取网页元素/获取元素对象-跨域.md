# 描述
根据iframe对象和xpath，跨域获取元素对象

![](https://cdn.nlark.com/yuque/0/2023/png/22242030/1688348573743-a44f9188-a924-4fae-9958-163d0bb82f97.png)



# 参数配置
## 指令输入
### 常规
**iframe对象：**通过【初始化iframe】指令或者【切换iframe】指令获取的输出对象

**xpath：**目标元素xpath，支持传入数组，可以实现精确切入多层iframe查找元素。传入数组，则不会全局查找

**基于当前iframe全局查找：**

**勾选：**会基于当前iframe对象以及所有子iframe对象查找元素

**不勾选：**只在当前iframe对象查找元素



## 指令输出
**保存网页元素至：**查找到的网页元素
