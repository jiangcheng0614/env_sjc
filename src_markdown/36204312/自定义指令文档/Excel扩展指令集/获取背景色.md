# 一.功能描述
<font style="color:rgb(51, 51, 51);">获取指定单元格的背景颜色，返回该颜色三原色数值（R，G，B)；单元格未设置背景颜色则返回None。</font>

![](https://cdn.nlark.com/yuque/0/2023/jpeg/1675452/1676891161603-f8cfd7ae-4dc5-4ac0-b2bc-31e0d7b68c44.jpeg)



# 二.<font style="color:rgb(51, 51, 51);">配置项说明</font>
### 输入
**<font style="color:rgb(51, 51, 51);">Excel文件：</font>**

<font style="color:rgb(51, 51, 51);">设置</font>输入需要操作的Excel文件路径，也可使用右边选择文件弹框输入

**<font style="color:rgb(51, 51, 51);">工作表：</font>**

输入需操作的Sheet页工作表，默认值为0，即第一个Sheet页

_<font style="color:rgb(102, 102, 102);">注：</font>_  
_<font style="color:rgb(102, 102, 102);">1.默认为index方式（数字形式），默认值为数字0，代表了Excel文件中的第一个Sheet页，填入相应的数字代表了相应的Sheet页；</font>_  
_<font style="color:rgb(102, 102, 102);">2.可使用真实的Sheet页名称（字符串形式），如‘测试1’，需要注意区分大小写，必须一模一样，注意空格；</font>_  
_<font style="color:rgb(102, 102, 102);">3.可填入变量代替。</font>_

**单元格：**

输入需操作的单元格编号，默认值'A1'，即指定需要获取背景色的单元格

_<font style="color:rgb(102, 102, 102);">注：</font>_  
_<font style="color:rgb(102, 102, 102);">1.默认字符串形式，填入单元格名称；</font>_  
_<font style="color:rgb(102, 102, 102);">2.可使用变量代替。</font>_

**打开模式：**

选择打开模式，默认为应用

### <font style="color:rgb(51, 51, 51);">输出</font>
返回背景色值**<font style="color:rgb(51, 51, 51);">：</font>**

返回该颜色三原色数值（R，G，B)

例：返回 (255, 0, 0)  为红色背景色
