# 参数配置
## 指令输入
### 常规
**指令名称/指令编码：**必填，字符串，选择自定义指令的名称，或者编码。

**检索类型：**必填，字符串，选择检索指令的方式

**子指令名称：**必填，字符串，选择要调用的子指令名称

**输入参数：**选填， 如果子指令有参数，需要以字典的方式传参 

### 高级
无



## 指令输出
**保存返回值至：**保存指令返回结果
