# 参数配置
## 指令输入
### 常规
**选择Python模块：**选择Python模块

**选择函数：**选择要调用的函数

**输入参数：**如果函数有参数，需要以字典的方式传参 

### 高级
**__file__：**保持默认值__file__,不要修改



## 指令输出
**保存返回值至：**保存函数返回结果
